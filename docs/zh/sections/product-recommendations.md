# 商品推荐（Related products）

Related products（商品推荐）分区用于根据顾客当前浏览的商品，动态展示相关商品或互补商品。它适合放在商品详情、规格比较等内容之后，帮助顾客继续浏览同类选择，或发现能够搭配购买的商品。

![Related products 设置面板与前台效果](/images/zh/product-recommendations.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开默认商品模板或需要编辑的商品模板。
4. 点击 **添加分区**，选择 **Related products** 或对应的商品推荐分区。
5. 设置推荐类型、标题、描述和商品数量。
6. 完成设置后，点击右上角的 **保存**。

此分区会读取当前商品上下文并自动生成推荐内容，不需要手动添加商品内容块。主题编辑器中的预览结果会随当前预览商品和推荐类型而变化。

## Recommendation type

设置需要展示的商品推荐类型：

- **Related products**：展示与当前商品相关的其他商品，适合提供同类选择并鼓励顾客继续浏览。
- **Complementary products**：展示适合与当前商品搭配购买的互补商品，适合配件、组合用品或交叉销售场景。

截图中选择 **Related products**。切换类型后，应检查推荐结果是否符合该商品的购买场景；如果某种类型没有可用推荐，前台可能显示较少商品或不显示商品列表。

## Color scheme

选择当前分区使用的全局配色方案，影响分区背景、标题、描述、商品信息和浏览控件等元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用同一方案的分区，因此保存前应检查文字、价格、箭头及商品图片与背景之间的对比度。

## Heading

### Heading

设置分区主标题。该字段支持富文本格式、链接和动态来源。截图中的标题为 **Related Product**。

标题应与 **Recommendation type** 保持一致，例如：

- 使用 **Related products** 时，可填写 `Related products`、`You may also like` 或“猜你喜欢”。
- 使用 **Complementary products** 时，可填写 `Complete the look`、`Frequently bought together` 或“搭配购买”。

### Heading size

设置分区主标题的字号。截图中选择 **2XL**，下拉菜单提供以下选项：

- **4XL、3XL、2XL、XL**：主题预设的响应式字号层级。
- **H1、H2、H3、H4、H5、H6**：采用对应标题层级的主题样式。

较大的字号能强化推荐区域，但也更容易在移动端换行。商品页面通常已经有一个主要商品标题，因此推荐分区应保持次要视觉层级；截图中的 **2XL** 适合较醒目的分区标题。

### Description

设置标题旁或标题下方的辅助说明。该字段支持富文本格式、链接、列表和动态来源。

截图中的示例文案为 **Personalized with AI insight. Personalized with AI insight. Personalized with AI insight.**。实际使用时建议改为简短、有信息量的说明，例如“根据你正在浏览的商品，为你推荐更多相似选择”。

桌面端中，描述可能与标题分列显示；文字过长会增加标题区域高度，因此应同时检查窄屏下的换行效果。

## Subtitle

### Subtitle size

设置商品卡片辅助文字或副标题的字号。截图中的值为 **XL**。

调整时应检查商品卡片中的品牌名、辅助信息与商品标题之间是否层级清晰。字号过大可能使较长文字换行，并压缩价格或导航控件的空间。

### Products limit

设置分区最多展示的推荐商品数量。截图中的值为 `4`，前台对应显示四张商品卡片。

该数值是展示上限；实际数量取决于当前商品可用的推荐结果。增加数量后，应检查页面长度、卡片宽度以及浏览控件是否仍然清晰；减少数量则可让推荐区域更紧凑。

## Padding

- **Padding Top**：设置分区顶部留白，单位为 `px`。截图中的值为 `32 px`。
- **Padding Bottom**：设置分区底部留白，单位为 `px`。截图中的值为 `32 px`。

上下留白通常可以保持一致。如果商品详情或相邻分区本身已有较大间距，可适当减小数值，避免内容之间出现过宽的空白区域。

## 商品展示说明

推荐商品的图片、名称、价格和链接来自 Shopify 中的商品数据。截图中的商品卡片还显示品牌文字和浏览箭头；这些内容的实际显示效果取决于当前主题的商品卡片样式及可用的推荐结果。

如果前台没有出现预期商品，请检查：

- 当前页面是否为有效的商品模板并具有商品上下文。
- 被推荐商品是否已发布到在线商店销售渠道。
- 推荐类型下是否存在可用的推荐结果。
- **Products limit** 是否设置为合适的数量。

## 发布前检查

1. 分别切换 **Related products** 与 **Complementary products**，确认所选类型符合销售目的。
2. 使用不同商品预览模板，检查推荐结果是否会随当前商品正确变化。
3. 确认标题和描述准确、简短，并与推荐类型保持一致。
4. 检查主标题字号、副标题字号及长文案在移动端的换行效果。
5. 确认商品图片、名称、价格和链接正确，且商品均可正常访问。
6. 检查实际商品数少于 **Products limit** 时的布局。
7. 测试商品卡片及左右浏览控件的鼠标、触摸和键盘操作。
8. 确认当前配色方案下的文字、价格和控件具有足够对比度。
9. 检查上下留白与商品详情、比较表及后续分区是否协调。
10. 点击 **保存**，并在桌面端和移动端分别完成一次商品访问测试。

::: tip 建议
相关商品适合帮助顾客比较同类选择，互补商品则更适合推动搭配购买。标题和描述应明确传达当前推荐逻辑，避免让顾客误以为推荐商品已经包含在当前商品中。
:::
