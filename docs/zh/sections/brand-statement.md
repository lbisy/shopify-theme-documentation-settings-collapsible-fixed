# 品牌宣言（Brand Statement）

Brand Statement（品牌宣言）分区用于以大字号文字集中展示品牌理念、价值主张或核心承诺。分区结构简洁，可配合笔刷装饰图增强视觉重点，适合放在关于我们页面、品牌故事页面或首页内容之间。

![Brand Statement 设置面板与前台效果](/images/zh/brand-statement.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Brand Statement**。
5. 填写品牌宣言，并按需上传笔刷装饰图。
6. 调整字号、配色和上下间距。
7. 点击右上角的 **保存**。

此分区不需要添加内容块，所有内容都在分区设置面板中完成。

## Color scheme

选择当前分区使用的全局配色方案，影响背景、宣言文字和装饰元素的整体呈现。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用该方案的分区，因此保存前应确认大号文字与背景之间具有足够的对比度。

## Statement text

设置需要在前台展示的品牌宣言。该字段支持文字样式、加粗、斜体、链接、项目符号和编号列表，并可连接动态来源。

品牌宣言应聚焦一个清晰观点，建议使用一至三句简洁文案说明品牌为何存在、重视什么以及能够为顾客带来什么。避免堆叠空泛形容词、复杂促销条件或与品牌定位无关的信息。

使用富文本格式时应注意：

- 只强调最重要的词语，避免整段加粗。
- 链接应指向与宣言直接相关的品牌故事、材料说明或服务承诺页面。
- 列表会改变大字号宣言的视觉节奏，仅在内容确实需要分项时使用。
- 长句在窄屏设备上可能产生多行换行，应优先精简文字。

## Heading

### Heading size

设置品牌宣言的字号。较大的字号能够形成视觉焦点，但也会显著增加分区高度。

调整后应检查桌面端与移动端的换行位置，避免单字、标点或过短词组独占一行。中文与英文版本的字符长度差异较大，多语言店铺应分别预览每种语言。

## Brushstroke decoration image

选择显示在宣言文字附近的笔刷装饰图片，也可以连接符合条件的动态来源。

建议使用背景透明、边缘清晰的 PNG 或 WebP 图片，并让图形颜色与当前配色方案协调。装饰图不应遮挡文字，也不应包含必须被顾客读取的活动规则、价格或品牌声明。

若不需要装饰效果，可以将该字段留空。上传图片后应重点检查：

- 图形在浅色和深色配色下是否清晰。
- 图形是否覆盖文字或降低文字对比度。
- 移动端缩放和裁切是否自然。
- 图片文件是否经过压缩，避免只为装饰效果增加过多页面体积。

## Padding

- **Padding Top**：设置分区顶部与上一个分区之间的留白，单位为 `px`。
- **Padding Bottom**：设置分区底部与下一个分区之间的留白，单位为 `px`。

可先让上下间距保持一致，再根据宣言行数和相邻分区的背景进行调整。大字号文案本身会占用较多空间，移动端不宜设置过大的上下留白。

## 自定义 CSS

展开 **自定义 CSS** 可以为当前分区添加局部样式。适合对默认设置未覆盖的细节进行小范围调整，例如限制文本宽度或微调装饰图位置。

自定义规则应限制在当前分区范围内，避免影响其他分区或全局富文本样式。修改后需在桌面端、移动端和主题编辑器中分别验证；主题升级后也应重新检查规则兼容性。

## 文案建议

- 先明确一个核心主张，再用具体词语说明品牌差异。
- 使用顾客能够理解的表达，减少行业术语和无法验证的绝对化承诺。
- 宣言应与商品、品牌故事和实际服务一致，避免只追求视觉效果。
- 环保、功效、质量或行业领先等声明应有事实依据，并符合销售地区的广告要求。
- 不要仅通过颜色、装饰图或文字样式传递关键信息。
- 多语言内容应进行人工校对，不要直接沿用长度和语序不自然的逐字翻译。

## 配置示例

| 设置项 | 示例值 |
| --- | --- |
| Color scheme | 与页面背景有明显对比的方案 |
| Statement text | `我们以透明选材和耐用设计，创造值得长期使用的产品。` |
| Heading size | `2XL` |
| Brushstroke decoration image | 透明背景的品牌笔刷图形 |
| Padding Top / Bottom | `32 px / 32 px` |

## 发布前检查

1. 确认 Statement text 已替换为实际品牌内容，没有无意义的默认文字。
2. 检查文案语法、标点和事实表述，并确认品牌承诺可以被验证。
3. 分别在桌面端和移动端检查大字号文字的换行、对齐和分区高度。
4. 检查普通、加粗、斜体及链接文字与背景之间的对比度。
5. 如果添加了链接，使用鼠标和键盘实际测试链接目标及焦点状态。
6. 如果上传了装饰图，确认其不会遮挡文字，并测试浅色、深色和窄屏状态。
7. 检查装饰图片的文件大小，避免影响页面加载速度。
8. 确认上下间距与前后分区衔接自然，没有过度留白或视觉拥挤。
9. 检查自定义 CSS 不会影响页面上的其他分区。
10. 点击 **保存**，再从实际店面完成桌面端和移动端浏览测试。
