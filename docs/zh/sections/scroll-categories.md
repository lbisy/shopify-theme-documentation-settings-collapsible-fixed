# 滚动分类（Scroll Categories）

Scroll Categories（滚动分类）用于以大字号列表展示某个商品系列中的商品。分区顶部可显示系列标题和说明，中部展示商品名称、价格等信息，底部可添加补充文案，适合营造具有视觉节奏的分类浏览区域。

![Scroll Categories 设置面板与前台效果](/images/zh/scroll-categories.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Scroll Categories**。
5. 在 **Select Collection** 中选择需要展示的商品系列。
6. 配置文字、字号和商品数量后，点击右上角的 **保存**。

此分区直接读取所选商品系列，不需要逐个添加商品内容块。商品标题、价格和链接来自 Shopify 中的商品数据。

## Color Scheme

选择当前分区使用的全局配色方案，影响背景、标题、说明、商品名称、价格及其他文字元素。深色背景与浅色文字的组合可以形成较强对比。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用同一方案的分区，保存前应检查普通文字、链接及交互状态与背景之间的对比度。

## Font

面板中的 **Font** 为字体相关设置区域。若当前主题版本未显示独立选项，分区会沿用主题的全局字体设置；可在 **主题设置 → Typography** 中调整全站标题和正文字体。

## Section heading

### Heading size

设置滚动商品列表中主要标题文字的字号，例如商品名称。可从 **2XL** 开始预览。

较大的字号可以增强视觉冲击力，但长商品标题可能会被截断或换行。调整后应使用短标题和长标题商品分别测试桌面端与移动端。

## Subtitle

### Subtitle size

设置列表中次级信息的字号，例如 **XL**。次级信息应与主要商品标题形成清晰层级，同时保证商品类型、价格等内容容易阅读。

## Collection title heading

### Heading size

设置分区顶部系列标题的字号，例如 **3XL**；标题可使用 **New Arrivals** 等简短文案。

该字号通常大于商品标题字号。系列标题较长时，应检查其是否与右侧说明文字重叠，或在移动端产生不自然的换行。

### Maximum items

设置从所选商品系列中最多显示的商品数量。例如设为 `4` 时，最多展示 4 件商品。

该数值是展示上限；如果系列中的可用商品不足，前台只会显示实际可用的商品。系列商品总数和实际显示数量可能不同。

### Heading

设置分区顶部显示的系列标题。该字段支持富文本格式、链接和动态来源，例如 **New Arrivals**。

此标题不必与所选 Shopify 商品系列名称相同，因此修改系列后也应检查标题是否仍与内容匹配。

### Description

设置系列标题旁的说明文字。该字段支持常用富文本格式、链接、列表和动态来源，例如 **Add a description to your category.**。

建议使用一到两句简短说明，介绍该系列的选品主题、使用场景或主要特点。过长内容会增加分区顶部高度，并可能削弱商品列表的视觉重点。

### Select Collection

选择需要在当前分区展示的 Shopify 商品系列，例如 **Chairs**。

分区会读取该系列内已发布到在线商店销售渠道的商品。若前台没有出现预期商品，请检查：

- 商品是否为启用状态并已发布到当前销售渠道。
- 商品是否符合自动系列的筛选条件，或已加入手动系列。
- **Maximum items** 是否限制了显示数量。
- 系列中的商品排序是否符合预期。

### Caption

设置分区底部的补充文案。该字段支持富文本格式、链接、列表和动态来源，例如 **Add a caption to your category.**。

Caption 可用于补充系列说明、活动信息或浏览提示。没有需要展示的内容时可以留空，避免出现无意义的占位文字。

## Padding

- **Padding Top**：设置分区顶部与上一个分区之间的留白，单位为 `px`，例如 `32 px`。
- **Padding Bottom**：设置分区底部与下一个分区之间的留白，单位为 `px`，例如 `32 px`。

上下留白通常可保持一致。由于此分区使用大字号排版，整体高度可能较高；如果与相邻分区之间显得过于松散，可适当减小 Padding。

## 前台内容与交互

分区由以下区域组成：

- 顶部系列标题、商品数量提示和 Description。
- 中部商品列表，包括次级标签、商品标题和价格。
- 底部 Caption 文案。

顾客滚动页面时可以依次浏览系列中的商品。商品名称应链接到对应商品详情页；价格和货币格式由商品数据、商店市场及当前货币设置决定。

::: tip 内容搭配
大字号商品列表更适合数量有限、名称简洁且彼此差异明显的商品。可先展示 `4` 件商品并根据页面长度调整；商品过多会显著拉长页面，也会降低每件商品的视觉重点。
:::

## 配置示例

以下是一组可直接参考的配置：

| 设置项 | 示例值 |
| --- | --- |
| Color Scheme | 深色方案 |
| Section heading size | `2XL` |
| Subtitle size | `XL` |
| Collection title heading size | `3XL` |
| Maximum items | `4` |
| Heading | `New Arrivals` |
| Description | `Add a description to your category.` |
| Select Collection | `Chairs` |
| Caption | `Add a caption to your category.` |
| Padding Top / Bottom | `32 px / 32 px` |

## 发布前检查

1. 确认 Heading、Description、Caption 与所选商品系列内容一致。
2. 检查系列商品数量不足、等于和超过 **Maximum items** 时的显示结果。
3. 确认商品标题、价格和链接来自正确商品，并使用正确的市场货币格式。
4. 使用包含长标题、促销价格或缺货状态的商品检查排版。
5. 在桌面端检查系列标题与 Description 不会重叠，商品名称与价格保持清晰对齐。
6. 在移动端检查大字号标题的换行、商品列表高度和触控区域。
7. 使用键盘 `Tab` 和 `Enter` 测试商品链接及文案中的链接。
8. 检查配色方案下所有文字和链接的对比度。
9. 点击 **保存**，并在实际店面完成一次从该分区进入商品详情页的流程。
