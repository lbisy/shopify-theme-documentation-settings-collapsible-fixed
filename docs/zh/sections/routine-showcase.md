# 场景商品展示（Routine Showcase）

Routine Showcase（场景商品展示）用于在大幅背景区域中组合商品、内容卡片和旋转徽章。它适合突出新品、主题系列、使用场景或品牌主张，并通过独立的位置设置控制各元素在画面中的布局。

![Routine Showcase 设置面板与前台效果](/images/zh/routine-showcase.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Routine Showcase**。
5. 上传背景图片并选择需要展示的商品系列。
6. 配置内容、链接及元素位置。
7. 点击右上角的 **保存**。

## Color Scheme

选择当前分区使用的全局配色方案，影响背景、文字、内容卡片、链接、图标和旋转徽章等元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用该方案的分区，因此应检查正文、行动链接和徽章与背景图片之间的对比度。

## Font

面板中的 **Font** 为字体相关设置区域。若当前主题版本未显示独立选项，分区会沿用主题的全局字体设置；可以在 **主题设置 → Typography** 中调整全站标题和正文字体。

## Section heading

### Heading size

设置内容主标题的字号或标题层级，例如 **H4**。应根据分区在页面中的信息层级选择大小，而不是只考虑视觉尺寸。

同一页面应保持清晰的标题结构。若页面已经有主标题，此分区通常使用较低层级的标题更合适。

### Background Image

选择分区使用的背景图片。也可以通过动态来源连接符合条件的图片字段。

建议使用清晰的横向大图，并在上传前保留足够的主体安全区域。由于桌面端和移动端的容器比例不同，图片边缘可能被裁切，重要人物、商品或文字不应紧贴边缘。

::: tip 图片选择
背景图片应为内容卡片和徽章保留相对简洁的区域。细节过多或明暗变化剧烈的图片，会降低叠加文字的可读性。
:::

### Select Collection

选择为该分区提供商品内容的 Shopify 商品系列，例如 **女士衣服**。

如果商品没有正常显示，请检查：

- 商品是否为启用状态并已发布到在线商店销售渠道。
- 商品是否已加入手动系列，或符合自动系列的筛选条件。
- 商品是否具有可用的主图和必要的商品信息。
- 系列中的商品排序是否符合展示需求。

### Product position

设置商品视觉元素在分区中的水平位置：

- **Left**：靠左显示。
- **Center**：居中显示。
- **Right**：靠右显示。

选择位置时应同时考虑背景图片主体和内容卡片的位置，避免商品被文字遮挡或与背景主体重叠。

### Content position

设置内容卡片在分区中的对齐位置，例如 **Center right**。可用位置以主题下拉菜单为准，通常由垂直方向与水平方向组合而成。

内容卡片包含 Eyebrow、Heading、Description 和行动链接。调整位置后，需要检查卡片与商品、徽章及分区边缘之间是否保留足够空间。

### Badge position

设置旋转徽章在分区中的位置，例如 **Bottom right**。

徽章属于辅助视觉元素，不应遮挡商品主体或主要行动链接。移动端空间较小时，尤其需要确认徽章不会与内容卡片重叠。

## 内容设置

### Eyebrow

设置主标题上方的短眉题，适合显示年份、系列标签或简短状态，例如 **2026©**。该字段支持基础富文本格式、链接和动态来源。

Eyebrow 应保持简短，并与主标题形成清晰层级。不需要眉题时可以留空。

### Heading

设置内容卡片的主标题，例如 **Personalized with AI insight.**。该字段支持基础富文本格式、链接和动态来源。

标题应直接表达系列价值或核心卖点。建议控制长度，以减少移动端出现过多换行。

### Description

设置主标题下方的说明文字。该字段支持富文本格式、链接、项目列表、编号列表和动态来源。

Description 可用于补充产品特点、适用场景或活动说明。内容较多时应使用短段落或列表提高可读性。

### Call to Action Text

设置行动链接显示的文字，例如 **See Our New Product**。该字段支持基础富文本格式和动态来源。

建议使用明确的动作词，如“查看新品”“探索系列”或“立即选购”，让顾客能够预判点击后的目标。

### Link URL

设置行动链接的目标地址，可选择商品系列、商品、页面或其他有效链接，例如 **所有产品系列**。

Call to Action Text 和 Link URL 应配合使用。只有文字而没有有效链接时，顾客可能无法完成跳转；只有链接而没有文字时，前台可能不会显示清晰入口。

## 旋转徽章

### Rotate Badge Text

设置旋转徽章外围循环显示的文字，例如 `PRODUCT GUARANTEE NATURAL`。

建议使用简短的大写词组，并在词组之间加入空格或分隔符。文字过长会变得拥挤，文字过短则可能出现较大的重复空隙。

### Rotate Badge Percent

设置徽章中心显示的文字，例如 `100%`。该字段是展示文案，不会自动计算商品评分、折扣或成分比例。

如果使用百分比或认证类文字，应确保内容真实、可验证，并符合商店所在地的广告和标签要求。

## 布局搭配

Product position、Content position 和 Badge position 共同决定分区布局：

| 商品位置 | 内容位置 | 徽章位置 | 适用效果 |
| --- | --- | --- | --- |
| Left | Center right | Bottom right | 商品与内容左右分离，阅读路径清晰 |
| Center | Bottom left | Top right | 商品作为中心主体，文字和徽章分置两侧 |
| Right | Center left | Bottom left | 适合背景主体偏右或从左向右阅读的版式 |

这些组合仅作为起点。最终位置应根据背景图片主体、商品轮廓和文字长度调整，并分别预览桌面端和移动端。

## 发布前检查

1. 确认背景图片清晰，桌面端和移动端裁切后主体仍然完整。
2. 检查所选系列中的商品状态、图片、排序和销售渠道发布状态。
3. 确认商品、内容卡片和旋转徽章没有相互遮挡。
4. 使用短标题、长标题和多段 Description 测试内容卡片高度。
5. 检查 Eyebrow、Heading、Description 和行动链接的文字对比度。
6. 点击行动链接，确认它打开正确的商品、系列或页面。
7. 观察旋转徽章的文字间距、中心文案和动画流畅度。
8. 使用键盘 `Tab` 和 `Enter` 检查链接可访问性，并确认焦点状态清晰。
9. 检查系统启用“减少动态效果”时的徽章体验。
10. 点击 **保存**，并在实际店面分别完成桌面端和移动端检查。
