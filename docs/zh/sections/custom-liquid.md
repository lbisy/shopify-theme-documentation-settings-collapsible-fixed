# 自定义 Liquid

此模块用于在页面中添加 **自定义 Liquid** 内容。

## 添加方法

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区 / Add section**。
5. 选择 **自定义 Liquid**。
6. 配置内容后点击 **保存**。
