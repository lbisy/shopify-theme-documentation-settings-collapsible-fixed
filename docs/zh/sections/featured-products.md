# 精选商品（Featured Products）

Featured Products（精选商品）分区用于将多个商品系列组织成标签页，并以商品网格或轮播形式集中展示。顾客可以切换标签查看不同系列，适合在首页呈现新品、畅销商品或主题选品。

![Featured Products 设置面板与前台效果](/images/zh/featured-products.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Featured Products**。
5. 展开分区并点击 **添加 Collection**。
6. 为内容块选择需要展示的商品系列，并重复添加其他系列。
7. 完成设置后，点击右上角的 **保存**。

左侧列表中的每个 **Collection** 内容块对应一个标签页，例如 **Kitchen**、**Chairs** 和 **new**；可以拖动内容块调整标签顺序。

## General

### Color Scheme

选择整个 Featured Products 分区使用的全局配色方案，影响分区背景、标签文字、商品信息、导航控件和按钮等元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用该方案的分区，保存前应检查普通、选中和悬停状态的对比度。

## Font

面板中的 **Font** 为字体相关设置区域。若当前版本未显示额外选项，分区文字将沿用主题的全局字体设置；全局标题和正文字体可在 **主题设置 → Typography** 中调整。

## Heading

### Heading size

设置顶部系列标签标题的字号，例如 **2XL**。较大的字号能突出系列名称，但当标签较多或名称较长时，也更容易占满一行，因此需要配合 **Desktop tab header behavior** 检查桌面端排版。

## Products Settings

### Maximum items

设置每个已选商品系列最多显示的商品数量，例如 `8`。

该数值是展示上限；如果某个系列中的可用商品少于上限，前台只会显示实际可用的商品。商品还需要处于可售状态并发布到当前在线商店销售渠道。

### Items per row on desktop

设置桌面端每行显示的商品数量，例如 `4`。列数越多，单张商品卡片越窄；商品标题较长或卡片信息较多时，可适当减少列数。

### Items per row on mobile

设置移动端每行显示的商品数量：

- **1**：每行显示 1 件商品，图片和商品信息空间更充足。
- **2**：每行显示 2 件商品，可在同一屏幕展示更多内容。

移动端使用两列时，应重点检查商品标题、价格、徽章和按钮是否发生重叠或过度换行。

### Show Navigation

控制是否显示商品浏览导航。开启后，顾客可以使用导航控件浏览当前可见范围之外的商品；关闭后界面更简洁。

当商品数量不超过当前布局可见数量时，即使开启导航，也可能没有额外内容可供切换。

### Show Pagination

控制是否显示分页或当前位置指示。开启后可帮助顾客判断当前浏览位置。

导航和分页可以根据展示需求单独启用。启用后应在桌面端与移动端分别检查其位置、可见性和点击区域。

### Show View All Button

控制是否显示“查看全部”按钮。开启后，顾客可以从当前标签进入对应商品系列页面。

启用前应确保每个 Collection 内容块都选择了有效系列，并测试按钮是否跟随当前选中的标签跳转到正确页面。

### Desktop tab header behavior

设置桌面端标签标题超出可用宽度时的排版方式。选择 **Wrap to multiple lines** 后，标签较多或名称较长时会换到下一行显示。

使用多行换行时，需要检查标签之间的间距和分区高度；调整为主题提供的其他行为时，则应确认所有标签仍然容易发现和操作。

## Brand

Brand 设置用于配置分区中的装饰性品牌跑马灯。它可以重复展示品牌文字，并通过移动方向、速度、间距和高度形成动态视觉背景。

### Brand Name

设置跑马灯中显示的品牌名称。该字段支持富文本格式、链接和动态来源，例如 **Esen**。

建议使用简短品牌名或短语。文字过长会减少重复次数，并可能在较窄屏幕上形成拥挤的视觉效果。

### Brand marquee direction

设置品牌文字的滚动方向。选择 **Right to left** 时，文字从右向左移动。选择方向时可结合页面语言阅读方向和其他动态元素进行判断。

### Pause marquee on hover

控制顾客将指针移到品牌跑马灯上时是否暂停动画。此功能仅在支持悬停的设备上生效。

开启悬停暂停可以让顾客有时间查看文字或点击其中的链接，同时也能降低持续动画对阅读的干扰。

### Brand marquee speed

设置品牌跑马灯的移动速度：

- **Slow**：缓慢移动，视觉效果更平稳。
- **Normal**：常规速度。
- **Fast**：快速移动，动态感更强。

速度过快会影响阅读，也可能分散顾客对商品的注意力。通常建议从 **Normal** 开始预览，再根据品牌文字长度调整。

### Brand marquee spacing

设置重复品牌文字之间的间距，单位为 `px`，例如 `64 px`。

较小的间距会让文字排列更密集，较大的间距则会增加留白。应避免相邻文字看起来连成一个词组。

### Brand marquee height

控制装饰性品牌带占用的垂直空间，单位为 `px`，例如 `380 px`；文字大小也会随高度产生轻微缩放。

较大的高度会强化品牌背景，但也会增加分区的整体高度。调整后应检查它与商品图片、标签标题和相邻分区之间的比例。

::: tip 动效建议
品牌跑马灯属于装饰性动效。建议保持文字简短、速度适中，并开启悬停暂停；还应检查系统“减少动态效果”设置下的浏览体验。
:::

## Padding

- **Padding Top**：设置分区顶部留白，单位为 `px`，例如 `32 px`。
- **Padding Bottom**：设置分区底部留白，单位为 `px`，例如 `32 px`。

上下间距通常可以保持一致。如果相邻分区本身已有较大留白，可适当减小数值，避免页面出现过宽的空白区域。

## Collection 内容块

每个 **Collection** 内容块用于指定一个需要展示的商品系列。系列名称通常会作为顶部标签标题，系列内符合条件的商品则显示在对应标签页中。

管理内容块时可以：

- 点击 **添加 Collection** 新增系列标签。
- 打开 Collection 内容块并选择或更换商品系列。
- 拖动内容块调整标签顺序。
- 使用 **移除区块** 删除不需要的系列标签。

建议将最重要或访问量最高的系列排在第一位，因为它通常是顾客进入分区后首先看到的标签。避免选择空系列或多个内容高度重叠的系列。

## 配置示例

以下是一组可直接参考的配置：

| 设置项 | 示例值 |
| --- | --- |
| Heading size | `2XL` |
| Maximum items | `8` |
| Items per row on desktop | `4` |
| Items per row on mobile | `1` |
| Show Navigation | 关闭 |
| Show Pagination | 关闭 |
| Show View All Button | 关闭 |
| Desktop tab header behavior | `Wrap to multiple lines` |
| Brand Name | `Esen` |

| Brand marquee speed | `Normal` |
| Brand marquee spacing | `64 px` |
| Brand marquee height | `380 px` |
| Padding Top / Bottom | `32 px / 32 px` |

## 发布前检查

1. 切换所有 Collection 标签，确认系列名称和商品内容正确。
2. 检查空系列、商品不足以及商品超过 **Maximum items** 时的显示结果。
3. 确认商品图片、标题、价格、徽章和商品链接正确。
4. 分别检查桌面端每行列数和移动端 1 列/2 列布局。
5. 标签较多时，确认桌面端换行行为不会遮挡商品内容。
6. 如果启用导航、分页或“查看全部”，逐一检查其状态和目标链接。
7. 测试品牌跑马灯的方向、速度、间距、悬停暂停和不同屏幕宽度。
8. 使用键盘 `Tab`、方向键和 `Enter` 检查标签、商品及导航控件是否可操作。
9. 确认配色方案下的标签选中状态、商品文字和按钮具有足够对比度。
10. 点击 **保存**，并在桌面端和移动端分别完成一次系列切换与商品访问流程。
