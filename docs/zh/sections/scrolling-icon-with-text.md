# 滚动图标与文字（Scrolling icon with text）

Scrolling icon with text（滚动图标与文字）分区用于让多组图标和短文字在横向内容带中循环移动。它适合展示商品类别、服务特点、品牌承诺或活动关键词，并可通过方向、速度、间距和尺寸控制整体节奏。

![Scrolling icon with text 设置面板与前台效果](/images/zh/scrolling-icon-with-text.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Scrolling icon with text**。
5. 展开分区并点击 **添加 Item**。
6. 为 Item 设置图标和文字，重复添加所需内容。
7. 完成设置后，点击右上角的 **保存**。

左侧列表中的每个 **Item** 对应一组图标与文字。内容不足以填满滚动带时，主题会重复排列已有 Item，以保持连续滚动效果。

## Color scheme

选择当前分区使用的全局配色方案，影响背景、文字、图标和分隔线等元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用同一方案的分区，因此需要检查文字、图标和分隔线与背景之间的对比度。

## Scrolling

### Direction

设置内容带的滚动方向。选择 **Right to left** 时，图标和文字从右向左移动；也可以在下拉菜单中选择主题提供的其他方向。

同一页面包含多个滚动分区时，建议保持一致方向，避免同时出现方向相反的动画而分散注意力。

### Speed

设置内容带的移动速度：

- **Slow**：缓慢移动，阅读时间更充足。
- **Normal**：常规速度，适合大多数场景。
- **Fast**：快速移动，动态感更强。

文字较长或 Item 数量较少时，优先使用 **Slow** 或 **Normal**。速度过快会降低可读性，也可能影响对动态内容敏感的顾客。

### Horizontal spacing

设置相邻 Item 之间的水平间距，单位为 `px`，例如 `40 px`。

间距过小会让图标和文字组相互拥挤；间距过大则会在滚动带中形成明显空档。应结合文字长度和图标宽度一起调整。

### Height

控制滚动行所占的垂直高度，单位为 `px`，例如 `120 px`。

增大高度可以提升分区的视觉分量，并为较大的图标和文字提供更多空间；减小高度可形成更紧凑的公告带效果。

### Content size

按照滚动带高度的比例缩放图标和文字，单位为 `%`，例如 `100%`。

该设置与 **Height** 相互影响：Height 决定内容带的垂直空间，Content size 决定图标和文字在该空间中的相对大小。调整其中一项后，应重新检查另一项。

### Pause on hover

控制顾客将指针悬停在滚动行上时是否暂停动画。此功能只在支持悬停的设备上生效。

建议开启悬停暂停，让顾客可以从容阅读内容；触摸设备没有持续悬停状态，因此还需要确保移动速度本身足够舒适。

::: tip 动效建议
滚动内容应简短、易读，并避免承载必须立即阅读的重要信息。如果内容涉及配送限制、价格条件或法律说明，应在静态页面区域同步提供完整文字。
:::

## Layout

### Show divider

控制是否显示分区分隔线。开启后可增强滚动带与相邻内容之间的边界；关闭后视觉衔接更简洁。

分隔线的颜色通常来自当前配色方案。浅色背景下应确认线条不会过淡，深色背景下也不要让线条对比过强。

## Padding

- **Padding top**：设置分区顶部留白，单位为 `px`，例如 `32 px`。
- **Padding bottom**：设置分区底部留白，单位为 `px`，例如 `32 px`。

Height 控制滚动带本身的高度，Padding 则控制滚动带与分区边缘或相邻分区之间的距离。若整体区域显得过高，应先判断需要调整的是内容带高度还是外部留白。

## Item 内容块

每个 **Item** 用于定义一组滚动内容。为获得清晰、统一的展示效果，建议：

- 使用风格一致、线条粗细相近的图标。
- 将文字控制在一个简短词组内，例如商品类别或服务特点。
- 保持图标与文字语义一致。
- 避免连续添加完全相同的 Item，主题会自动循环已有内容。

管理 Item 时可以：

- 点击 **添加 Item** 新增一组图标与文字。
- 打开 Item 修改其内容。
- 拖动 Item 调整滚动顺序。
- 使用 **移除区块** 删除单个 Item。

## 自定义 CSS

展开 **自定义 CSS** 可添加仅作用于当前分区的样式覆盖。建议优先使用现有的方向、速度、间距、高度和内容尺寸设置；添加 CSS 后，需要检查循环衔接、移动端宽度以及动画性能。

## 删除分区

点击面板底部的 **删除分区** 会移除整个 Scrolling icon with text 及其所有 Item。若只需要删除一组内容，请打开对应 Item 并使用 **移除区块**。

## 参考配置

| 设置项 | 示例值 |
| --- | --- |
| Direction | `Right to left` |
| Speed | `Normal` |
| Horizontal spacing | `40 px` |
| Height | `120 px` |
| Content size | `100%` |
| Pause on hover | 开启 |
| Show divider | 开启 |
| Padding top / bottom | `32 px / 32 px` |

## 发布前检查

1. 确认所有 Item 的图标和文字正确，排列顺序符合预期。
2. 观察多个循环周期，确保首尾衔接自然，没有明显跳动或空白。
3. 测试 **Slow**、**Normal** 和 **Fast**，选择兼顾可读性与视觉节奏的速度。
4. 检查悬停暂停在桌面端是否生效，并确认触摸设备上的滚动速度舒适。
5. 分别检查桌面端和移动端的水平间距、内容大小与滚动带高度。
6. 确认分隔线、图标和文字在当前配色方案下清晰可见。
7. 检查系统启用“减少动态效果”时的浏览体验。
8. 点击 **保存**，并在实际店面再次确认动画性能与页面滚动流畅度。
