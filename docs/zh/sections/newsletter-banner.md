# 邮件订阅横幅（Newsletter Banner）

Newsletter Banner（邮件订阅横幅）分区用于展示标题、辅助文案和邮箱订阅表单，可搭配背景图片及局部模糊遮罩。它适合放在首页、博客页面或长内容页面末尾，引导顾客订阅新品、内容更新或营销邮件。

![Newsletter Banner 设置面板与前台效果](/images/zh/newsletter-banner.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Newsletter Banner**。
5. 填写标题、辅助标题和邮箱输入提示。
6. 按需上传背景图片并配置局部模糊遮罩。
7. 调整上下间距后，点击右上角的 **保存**。

此分区不需要添加内容块，订阅表单会随分区自动显示。

## Color Scheme

选择当前分区使用的全局配色方案，影响背景、标题、辅助标题、输入框、提交按钮和反馈信息等元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用该方案的分区，因此应检查普通文字、输入提示、按钮、焦点状态和错误信息与背景之间是否具有足够的对比度。

## Font

面板中的 **Font** 为字体相关设置区域。分区内文字会沿用主题的字体配置；如需调整全站字体，可前往 **主题设置 → Typography**。

## Heading

### Heading size

设置订阅横幅主标题的字号。主标题应突出订阅能够为顾客带来的价值，长标题需检查移动端换行和表单之间的间距。

### Heading

设置订阅横幅主标题。该字段支持基础富文本格式、链接和动态来源。

建议使用简短、真实的表达，例如“订阅新品与内容更新”。不要承诺并不存在的优惠、赠品或固定发送频率。

## Subtitle

### Subtitle size

设置辅助标题的字号。辅助标题应弱于主标题，但仍需在背景图片和模糊遮罩上保持清晰。

### Subtitle

设置主标题附近的辅助文案。该字段支持基础富文本格式、链接和动态来源。

可用于说明订阅内容、发送频率或退出方式。不需要辅助说明时可以留空，避免保留无实际意义的默认文字。

## Background Image

选择订阅横幅的背景图片，也可以连接符合条件的动态来源。

建议使用构图简洁、留有文字安全区域的高分辨率图片。图片中的重要主体应避开标题和邮箱表单所在区域，并分别检查桌面端和移动端裁切。

若不需要图片，可以将该字段留空，使用 Color Scheme 提供的纯色背景。

## Background image fit

设置背景图片如何适配横幅容器。

选择 **Cover (fill frame)** 时，图片会填满整个区域，容器不会出现明显空白，但图片边缘可能被裁切。若主题提供其他模式，保留完整图片时可能出现留白或未覆盖区域，应根据素材比例和页面设计选择。

## Background image position

设置背景图在容器中的对齐位置。选择 **Center** 时，以图片中心作为主要显示区域。

如果人物、商品或其他关键主体偏向图片一侧，应选择更合适的位置。调整适配方式后需要重新检查位置，因为裁切范围会随模式、容器尺寸和设备宽度变化。

## Input Placeholder

设置邮箱输入框内的提示文字，例如“请输入邮箱地址”。该字段支持动态来源。

占位文字应简短并与页面语言一致。它只用于提示输入内容，不应承担隐私说明、订阅条款或字段标签的全部作用；输入后占位文字会消失。

多语言店铺应为每种语言提供对应翻译，并检查较长文字不会与提交按钮重叠。

## Blur overlay

局部模糊遮罩用于弱化背景图片中位于文字和表单后方的细节，从而提升内容可读性。它不会替代正确的配色与文字对比度检查。

### Enable blur overlay

控制是否显示局部模糊遮罩。背景图细节较多或文字对比不足时可以开启；使用纯色背景或留白充分的图片时可以关闭。

### Blur radius

设置模糊强度，单位为 `px`。数值越大，遮罩范围内的背景细节越柔和。

过大的模糊值可能造成明显光晕并降低视觉清晰度，也可能增加浏览器渲染负担。应从较小数值开始调整。

### Blur width

设置模糊遮罩的宽度，使用百分比表示。应确保遮罩覆盖标题和表单，同时不要无意义地遮盖整张背景图。

### Blur height

设置模糊遮罩的高度，使用百分比表示。标题换行、辅助文案增加或表单反馈出现时，内容高度会变化，应为这些状态预留空间。

### Horizontal blur position

设置模糊遮罩的水平位置，使用百分比表示。`50%` 通常表示位于横幅水平方向的中间区域；向较小或较大数值调整可使遮罩向两侧移动。

### Vertical blur position

设置模糊遮罩的垂直位置，使用百分比表示。`50%` 通常表示位于横幅垂直方向的中间区域。

修改遮罩尺寸或位置后，应重新检查桌面端、平板和移动端。遮罩必须跟随文字与表单区域，而不是只在单一预览宽度下对齐。

## Padding

- **Padding Top**：设置分区顶部与上一个分区之间的留白，单位为 `px`。
- **Padding Bottom**：设置分区底部与下一个分区之间的留白，单位为 `px`。

可先让上下间距保持一致，再根据背景图比例、标题行数和表单反馈区域调整。移动端不宜使用过大的留白，以免订阅表单离开首屏或与正文相距过远。

## 自定义 CSS

展开 **自定义 CSS** 可以为当前分区添加局部样式，例如限制文案宽度或微调表单与模糊遮罩的位置。

自定义规则应限制在当前分区范围内，避免影响其他表单和输入框。修改后需测试空白、输入中、焦点、提交成功和提交失败等状态；主题升级后也应重新检查兼容性。

## 订阅与隐私建议

- 清楚说明订阅者将收到的内容，不使用误导性优惠承诺。
- 在适当位置提供隐私政策入口，并说明邮箱地址的用途。
- 根据目标市场要求取得电子邮件营销同意，不预先替顾客作出同意选择。
- 实际测试提交后的成功和错误提示，不能只确认输入框外观。
- 使用已订阅邮箱和未订阅邮箱测试，并在 Shopify 后台核对记录及营销状态。
- 邮箱的实际保存位置、订阅状态和后续自动化流程取决于主题表单实现、Shopify 设置及所使用的营销应用。
- 定期清理无效地址，并为每封营销邮件提供清晰的取消订阅方式。

## 配置示例

| 设置项 | 示例值 |
| --- | --- |
| Color Scheme | 与背景图有明显对比的方案 |
| Heading size | `XL` |
| Heading | `订阅新品与内容更新` |
| Subtitle size | `3XL` |
| Subtitle | `第一时间获取品牌动态` |
| Background Image | 留有中央文字区域的品牌图片 |
| Background image fit | `Cover (fill frame)` |
| Background image position | `Center` |
| Input Placeholder | `请输入邮箱地址` |
| Enable blur overlay | 开启 |
| Blur radius | `12 px` |
| Blur width / height | `60% / 80%` |
| Horizontal / Vertical blur position | `50% / 50%` |
| Padding Top / Bottom | `32 px / 32 px` |

## 发布前检查

1. 确认 Heading、Subtitle 和 Input Placeholder 已替换为实际内容，并完成多语言翻译。
2. 检查背景图片清晰、拥有合法使用权且关键主体没有被裁切或遮挡。
3. 分别测试背景图适配方式和位置在桌面端与移动端的效果。
4. 开启模糊遮罩时，检查半径、宽度、高度及位置能够覆盖文字与表单区域。
5. 检查标题、辅助标题、输入提示、按钮和反馈信息与背景之间的对比度。
6. 使用空值、格式错误邮箱和有效邮箱测试表单验证。
7. 提交有效邮箱后，确认成功提示清晰，并在 Shopify 后台核对订阅记录和营销状态。
8. 测试重复订阅、网络错误及其他失败状态，确保顾客能够理解下一步操作。
9. 使用键盘 `Tab`、`Shift+Tab` 和 `Enter` 操作输入框与提交按钮，确认焦点状态清晰可见。
10. 在移动端检查键盘弹出后的表单位置、按钮触控区域和页面滚动。
11. 检查隐私政策、营销同意文案及取消订阅流程符合目标市场要求。
12. 确认自定义 CSS 不会影响页面上的其他表单。
13. 点击 **保存**，再从实际店面完成一次完整订阅流程。
