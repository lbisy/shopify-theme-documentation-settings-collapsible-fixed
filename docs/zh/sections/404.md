# 404 页面

404 页面会在访客打开不存在或已删除的链接时显示。除了错误提示和返回首页按钮，还可以通过右侧的 **Contact Us** 设置完善联系区域，帮助访客继续咨询。

## 打开 404 页面设置

1. 进入 Shopify 后台的 **在线商店 > 主题**。
2. 找到当前主题，点击 **自定义**。
3. 使用顶部页面选择器打开 **404 页面**。
4. 在左侧模板列表中选择对应的 404 页面分区。
5. 完成设置后点击右上角的 **保存**。

## 配色与表单

### Color scheme

选择该区域使用的全局配色方案。修改方案本身时，请进入 **主题设置 > Colors**。

### Form background color

设置联系表单的背景颜色。建议让表单与页面背景有适当区分。

### Form text color

设置表单中的文字颜色，包括字段文字和相关提示。请确保它与表单背景具有清晰对比。

## 图片

### Image

为联系区域选择图片。可以上传新图片，也可以从 Shopify 文件库或免费图片中选择。

### Image alt text

填写图片替代文字，供屏幕阅读器使用，并在图片无法加载时显示。应简短描述图片内容；纯装饰图片可以留空。

## 文字内容

### Heading

设置联系区域的标题，例如“联系我们”或“需要帮助？”。编辑器支持加粗、斜体和链接等基本格式。

### Heading size

选择标题大小。较大的标题更醒目，但需要检查移动端是否出现过多换行。

### Description

填写标题下方的说明文字，可用于引导访客描述问题、咨询商品或联系客户服务。

### Button text

设置表单提交按钮的文字，例如“发送”或“提交信息”。建议使用简短、明确的操作词。

### Success message

设置表单成功提交后显示的提示，例如“感谢你的留言，我们会尽快回复”。提示应明确告诉访客信息已经提交成功。

### Watermark text

设置页面中的大号装饰文字。建议使用简短的品牌名称，并检查它不会影响主要标题、描述和表单的阅读。

## Padding

- **Padding Top**：调整该区域顶部的留白。
- **Padding Bottom**：调整该区域底部的留白。

增加数值会让内容更加舒展，减小数值则会让页面更紧凑。请分别检查桌面端和移动端效果。

## 保存前检查

- 404 标题和返回首页入口清晰可见。
- 图片和 Watermark 不会遮挡主要内容。
- 表单文字与背景颜色容易阅读。
- 标题和描述在移动端没有异常换行。
- 提交按钮文字明确。
- 测试提交表单后，Success message 能够正常显示。
- 上下间距与 Header、Footer 保持协调。

::: tip 建议
404 页面应帮助访客快速返回商店或取得联系。避免放置过多内容，优先保留返回首页、继续浏览和联系支持等明确操作。
:::
