# 分类精选展示（Category grid）

Category grid（分类网格）用于以图片卡片组成分类入口，帮助顾客快速进入不同的商品系列或内容页面。分区支持独立设置标题、说明、“查看全部”链接、桌面端与移动端网格，以及卡片图片样式。

![Category grid 设置面板与前台效果](/images/zh/category-grid.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Category grid**。
5. 点击 **添加 Item** 创建分类卡片。
6. 完成设置后，点击右上角的 **保存**。

左侧列表中的每个 **Item** 对应一张分类卡片。可以拖动 Item 调整顺序，也可以打开某个 Item 编辑其图片、文字和跳转链接。

## Color scheme

**Color scheme** 用于选择当前分区使用的全局配色方案，影响背景、标题、副标题、链接文字、箭头和卡片相关元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用同一方案的分区，因此保存前应检查文字和链接与背景之间的对比度。

## Heading

### Heading

设置分区主标题。该字段支持富文本格式和动态来源，例如 **Shop by category**。

### Heading size

设置主标题字号，例如 **2XL**。字号越大，标题越醒目，但也更容易在移动端换行；标题较长时应同时预览窄屏效果。

### Subheading

设置主标题下方的辅助说明。该字段支持常用富文本格式、链接、列表和动态来源，例如 **Explore collections curated for your routine.**。

### Subheading size

设置辅助说明的字号，例如 **Large**。建议让副标题的视觉层级低于主标题，并避免使用过长的段落。

### View all label

设置“查看全部”入口显示的文字，例如 **View all**。如果不需要显示该入口，可将标签留空。

### View all link

设置“查看全部”入口的目标链接，可选择商品系列、商品、页面或其他有效地址，例如 **所有产品系列**。

只有标签而没有有效链接时，入口可能无法完成跳转。配置后请在预览区域实际点击验证。

### Text alignment

控制标题、副标题和“查看全部”入口的对齐方式：

- **Left**：左对齐。
- **Center**：居中对齐。
- **Right**：右对齐。

此设置控制分区标题区域的排版。卡片内部文字的具体显示方式以主题实现和卡片内容为准。

## Grid

### Desktop columns

设置桌面端每行显示的卡片列数。例如设为 `4` 时，每行排列 4 张卡片。

列数越多，单张卡片可用宽度越小。分类名称较长或图片细节较多时，建议减少列数并检查标题换行。

### Desktop rows

设置桌面端网格的行数。例如设为 `2` 并与 4 列组合时，预览区域形成 4 列 × 2 行的布局。

添加的 Item 数量应与预期网格规模相匹配。调整行数后，需确认所有需要展示的分类都能按预期出现。

### Mobile columns

设置移动端每行的卡片数量：

- **1 column**：每行 1 张，图片和文字空间更充足。
- **2 columns**：每行 2 张，可在较短页面内展示更多分类。

移动端使用 2 列时，建议使用简短分类名称和主体居中的图片，避免文字过度换行或图片主体过小。

### Gap

设置卡片之间的横向和纵向间距，单位为 `px`，例如 `40 px`。

间距过小会使卡片显得拥挤；间距过大则可能减少首屏可见内容。调整时应同时查看桌面端多列布局和移动端两列布局。

## Card

### Image shape

设置分类图片容器的形状。选择 **Circle** 时，图片会以圆形区域显示。

使用圆形时，建议上传主体位于中央、四周留有安全空间的图片，避免重要内容在裁切后缺失。所有 Item 尽量使用一致的图片比例和构图风格。

### Image frame style

设置图片容器的外观：

- **Soft**：使用柔和的层次或阴影效果。
- **Border**：显示清晰边框。
- **None**：不添加额外边框样式。

选择样式时应结合当前配色方案检查边界是否清晰，尤其要留意浅色图片与浅色背景的组合。

### Show arrow

控制分类卡片标题旁是否显示跳转箭头。开启后，每个分类标题后方都会显示箭头，以强化卡片能够点击的提示；关闭后界面更简洁。

## Padding

- **Padding top**：设置分区内容与上一个分区之间的顶部留白，单位为 `px`，例如 `32 px`。
- **Padding bottom**：设置分区内容与下一个分区之间的底部留白，单位为 `px`，例如 `32 px`。

上下留白通常可以保持一致。如果相邻分区自身已经包含较大间距，可适当减少 Padding，避免页面出现过宽的空白区域。

## Item 内容块

点击左侧的某个 **Item** 可编辑单张分类卡片。为每个 Item 配置合适的图片、显示文字和目标链接，并确保图片内容与目标分类一致。

管理 Item 时可以：

- 点击 **添加 Item** 新增分类卡片。
- 拖动 Item 调整卡片顺序。
- 使用 **移除区块** 删除不需要的单张卡片。

建议优先排列访问量高、业务重要或顾客最常寻找的分类。不要让多个卡片使用相同标题却跳转到不同地址。

## 自定义 CSS

如果面板中显示 **自定义 CSS**，可展开并添加仅作用于当前分区的样式覆盖。建议优先使用分区已有的网格、卡片和间距设置；添加 CSS 后，应重新检查不同列数、长标题以及移动端布局。

## 配置示例

以下是一组可直接参考的配置：

| 设置项 | 示例值 |
| --- | --- |
| Heading | `Shop by category` |
| Heading size | `2XL` |
| Subheading | `Explore collections curated for your routine.` |
| Subheading size | `Large` |
| View all label | `View all` |
| View all link | 所有产品系列 |
| Text alignment | `Left` |
| Desktop columns / rows | `4 / 2` |
| Mobile columns | `2 columns` |
| Gap | `40 px` |
| Image shape | `Circle` |
| Image frame style | `Soft` |
| Show arrow | 开启 |
| Padding top / bottom | `32 px / 32 px` |

## 发布前检查

1. 确认标题、副标题和“查看全部”入口的内容及链接正确。
2. 检查每个 Item 的图片、名称与目标页面是否匹配。
3. 确认桌面端列数、行数和 Item 数量符合预期，没有异常空位。
4. 分别预览移动端 1 列和 2 列效果，检查图片裁切与文字换行。
5. 检查卡片间距、图片形状、边框样式和箭头是否保持一致。
6. 使用键盘 `Tab` 和 `Enter` 测试“查看全部”及所有卡片链接。
7. 确认当前配色方案下的标题、链接、箭头和图片边界清晰可见。
8. 点击 **保存**，并在桌面端和移动端分别测试所有分类入口。

::: tip 建议
分类卡片应尽量使用统一的图片比例、背景风格和命名规则。视觉一致的图片比单纯增加列数更能帮助顾客快速识别分类。
:::
