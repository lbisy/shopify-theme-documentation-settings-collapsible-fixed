# 关于我们（About Us）

**About Us** 分区适合用作关于我们页面的首屏介绍，通过配色、品牌图片、眉题、主标题和简短描述集中传达品牌定位。

![关于我们设置面板与前台效果](/images/zh/about-stats.svg)

## 添加方法

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开 About 页面所使用的模板，或切换到需要展示该内容的页面模板。
4. 点击 **添加分区 / Add section**，选择 **About Us**。
5. 配置图片和文字内容，并分别预览桌面端与移动端效果。
6. 点击右上角 **保存**。

## 配色与图片

- **Color scheme**：选择该分区使用的全局配色方案，控制背景、文字及相关元素的颜色。
- **编辑方案**：进入全局颜色设置，修改当前方案的背景色、文字色和强调色。修改后，其他使用同一方案的分区也可能同步变化。
- **Image**：上传品牌图片、产品氛围图或装饰图。图片位于文字内容后方，因此应避免使用细节过多、对比过强的素材，以免影响文字阅读。

若暂时不需要图片，可将 **Image** 留空，并通过纯色背景构建简洁的品牌首屏。

## 文字内容

- **Eyebrow text**：设置主标题上方的小号眉题，用于展示品牌名称或内容类别，例如 `About Eccenova`。支持加粗、斜体和链接等基础富文本格式。
- **Subtitle size**：调整辅助标题的字号，例如 **XL**。
- **Heading**：填写分区的核心标题，例如 `A Gentle Approach to Better Skin`。建议用一句简洁、有辨识度的文案概括品牌理念。
- **Heading size**：调整主标题字号，例如 **2XL**。
- **Description**：填写主标题下方的补充说明，例如品牌主张或价值说明。支持基础富文本、链接和列表格式。

主标题在桌面端可能显示为多行。建议避免过长文案，并在移动端检查断行是否自然、是否出现单字或短词独占一行。

## 图片替代文本

- **Image alt text**：填写图片的替代文本，用于无障碍访问以及图片无法加载时的说明，例如 `About Us`。

替代文本应准确描述图片内容或作用，不必重复页面标题，也不要堆砌关键词。若图片仅用于装饰且不传递额外信息，可根据主题的无障碍实现决定是否留空。

## Padding

- **Padding Top**：调整分区顶部留白，单位为 `px`，例如 `32 px`。
- **Padding Bottom**：调整分区底部留白，单位为 `px`，例如 `32 px`。

上下间距会影响首屏高度及其与前后分区的衔接。调整后应检查桌面端和移动端，避免内容过于拥挤或首屏留白过多。

## 配置示例

以下是一组完整的内容配置示例：

- **Eyebrow text**：`About Eccenova`
- **Subtitle size**：`XL`
- **Heading**：`A Gentle Approach to Better Skin`
- **Heading size**：`2XL`
- **Description**：`Because your skin deserves more than promises.`
- **Image alt text**：`About Us`
- **Padding Top / Bottom**：`32 px`

示例文案仅用于说明字段关系，正式发布时请替换为实际品牌内容。

## 使用建议

- 选择与品牌视觉一致的配色，并确保文字与背景具有足够对比度。
- 图片主体尽量避开标题所在区域，防止文字遮挡人物、商品或重要细节。
- 眉题保持简短，主标题突出一个核心主张，描述建议控制在一至两句话。
- 上传图片后填写有意义的替代文本，并检查图片加载失败时页面是否仍易于理解。
- 调整上下间距后，检查该分区与 Header 以及下一个内容分区之间的视觉衔接。
- 发布前分别预览桌面端和移动端，重点确认图片裁切、标题换行、文字对比度和首屏高度。
