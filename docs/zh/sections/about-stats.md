# 品牌介绍与数据（About Stats）

About Stats（品牌介绍与数据）分区用于组合品牌图片、年份或短标识、介绍文案、跳转链接和多项关键数据。它适合展示品牌历程、制造标准、服务成果、可持续承诺或企业规模等信息。

![About Stats 设置面板与前台效果](/images/zh/about-stats.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **About Stats**。
5. 上传图片，填写介绍文案和跳转链接。
6. 点击 **添加 Stat**，配置需要展示的数据项。
7. 点击右上角的 **保存**。

## Color scheme

选择当前分区使用的全局配色方案，影响背景、介绍文字、链接和数据项等元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用该方案的分区，因此应检查普通文字、大号数据和链接与背景之间是否具有足够的对比度。

## Font

面板中的 **Font** 为字体相关设置区域。分区内文字会沿用主题的字体配置；如需调整全站字体，可前往 **主题设置 → Typography**。

## Title

### Subtitle size

设置分区介绍文字中较大文本的字号。该选项位于 **Title** 设置区域；字号越大，越需要控制文案长度并检查移动端换行。

## Subtitle

### Heading size

设置分区次级文字的字号。该选项位于 **Subtitle** 设置区域，可与上方字号配合建立清晰的信息层级。

由于设置名称与实际视觉层级可能不同，调整时应以店面预览结果为准，并同时检查桌面端和移动端。

## Image

设置分区的主要图片，也可以连接符合条件的动态来源。适合使用品牌故事、产品细节、团队、工艺或门店环境图片。

建议上传清晰、高分辨率且主体明确的素材。重要内容应避开图片边缘，以降低响应式裁切造成的信息损失。图片只用于装饰时，不要把活动规则或关键文字直接嵌入图片。

## Secondary Image

设置第二张辅助图片，也可以连接符合条件的动态来源。辅助图片可用于补充不同视角、细节或场景，并与主图片形成视觉层次。

两张图片应保持相近的色调和内容主题，但不必完全相同。若只需要一张图片，可以将 **Secondary Image** 留空，并检查空缺状态下的布局是否仍然平衡。

## Year Text

设置分区顶部显示的年份、成立时间或简短标识。该字段支持动态来源。

建议控制为简短内容，例如成立年份、周年标识或阶段名称。该字段只显示输入的文字，不会根据当前年份自动更新；使用年份时，应在每次内容更新或主题发布前复核。

## Description

设置品牌介绍或核心主张。该字段支持富文本格式、链接、项目符号和编号列表，并可连接动态来源。

大字号排版更适合简洁内容。建议先用一至三句话说明品牌特点，再通过下方链接引导顾客查看完整故事。使用列表时，应检查其缩进和行距是否与整体布局协调。

## Link

设置文字链接的跳转目标。可以选择店铺内的商品、商品系列、页面或其他有效地址，也可以使用搜索选择目标。该字段支持动态来源。

外部地址需使用完整网址。链接目标应与介绍内容直接相关，例如品牌故事页面、商品系列或服务说明页面。

## Link Text

设置链接在前台显示的文字，例如“了解品牌故事”“查看新品”或“浏览全部系列”。该字段支持动态来源。

需要显示入口时，应同时填写 **Link** 和 **Link Text**；不需要入口时，可将两个字段同时留空。

## Stat 内容块

点击 **添加 Stat** 可以增加一项关键数据。每个 Stat 通常由简短标签和重点数值组成，例如“服务地区 / 12 个”或“可回收包装 / 90%”。

在左侧分区列表中可以拖动 Stat 调整顺序，也可以单独隐藏、复制或删除。选择某个 Stat 后填写对应的标签和数值；具体字段名称可能因主题版本而有所不同。

配置数据项时建议：

- 标签说明统计对象，数值突出可量化结果。
- 所有数据使用一致的格式，例如百分比、数量或年份不要随意混排。
- 数值较长时使用易读的缩写，但不要省略必要单位。
- 只使用真实、可核实且仍然有效的数据，必要时在目标页面补充统计口径。
- 控制数据项数量，避免移动端内容过长或单项宽度不足。

## Padding

- **Padding Top**：设置分区顶部与上一个分区之间的留白，单位为 `px`。
- **Padding Bottom**：设置分区底部与下一个分区之间的留白，单位为 `px`。

可先让上下间距保持一致，再根据图片高度、描述行数和 Stat 数量进行调整。移动端文字换行较多时，应避免过度压缩内容之间的间距。

## 内容与图片建议

- 主图负责建立场景，辅助图补充细节，避免两张图片传达完全重复的信息。
- 不要使用只有图片内文字才能理解的内容，以便多语言翻译和无障碍访问。
- Year Text、Description 和 Stat 应围绕同一个品牌主题，避免信息彼此无关。
- Description 说明品牌价值，Stat 提供证据，Link 引导顾客继续了解。
- 为有信息价值的图片配置准确的替代文本；纯装饰图片应按主题的无障碍实现处理。
- 涉及环保、质量、销量或用户数量的声明应可验证，并符合销售地区的广告要求。

## 配置示例

| 设置项 | 示例值 |
| --- | --- |
| Color scheme | 与页面背景有明显对比的方案 |
| Subtitle size | `3XL` |
| Heading size | `2XL` |
| Image | 品牌工艺主图 |
| Secondary Image | 产品细节辅助图 |
| Year Text | `创立于 2018 年` |
| Description | `从材料选择到包装设计，我们持续改进每一个环节。` |
| Link | 品牌故事页面 |
| Link Text | `了解品牌故事` |
| Stat 1 | `合作工坊 / 12 家` |
| Stat 2 | `可回收包装 / 90%` |
| Stat 3 | `服务地区 / 8 个` |
| Padding Top / Bottom | `32 px / 32 px` |

## 发布前检查

1. 确认 Image 和 Secondary Image 清晰、主题一致且拥有合法使用权。
2. 分别检查桌面端和移动端的图片裁切，确保关键主体完整可见。
3. 检查 Year Text 是否仍然有效，并移除无实际含义的默认内容。
4. 确认 Description 的富文本格式、换行和列表缩进没有破坏布局。
5. 实际点击文字链接，确认 Link 与 Link Text 均已填写且目标正确。
6. 核对每个 Stat 的标签、数值、单位和统计口径，确保信息真实一致。
7. 测试 Stat 数量较少和较多时的排列，并检查移动端是否出现拥挤或溢出。
8. 检查所有文字、大号数值和链接与背景之间的对比度。
9. 使用键盘 `Tab` 和 `Enter` 测试链接，确认焦点状态清晰可见。
10. 检查上下间距以及该分区与前后内容的衔接，保存后再从实际店面完成浏览和跳转测试。
