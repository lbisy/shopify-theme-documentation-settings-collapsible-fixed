# 商品系列商品列表（Collection）

Collection（商品系列商品列表）是商品系列模板的主要内容分区，用于展示当前商品系列中的商品，并提供筛选、排序、分页以及商品网格列数设置。

本文统一将 Shopify 的 `Collection` 翻译为“商品系列”，将 `Product grid` 翻译为“商品网格”，以便区分商品分组与商品排列区域。

![Collection 商品系列商品列表设置](/images/zh/collections.svg)

::: tip 数据来源
该分区会自动读取顾客当前访问的商品系列，不需要在主题编辑器中逐件选择商品。系列内的商品、初始排序和发布状态应在 Shopify 后台管理。
:::

## 使用前准备

1. 进入 **Shopify 后台 → 产品 → 商品系列**。
2. 创建或打开一个商品系列，填写名称和说明。
3. 选择商品系列类型：
   - **手动**：由管理员逐件添加商品。
   - **智能**：由 Shopify 根据条件自动加入符合要求的商品。
4. 设置系列内商品的默认排序方式。
5. 确认商品系列及其商品已发布到 **在线商店**。
6. 完善商品主图、标题、价格、库存、商品类型、供应商、标签、变体和相关元字段。

这些商品数据会影响卡片内容以及可供顾客使用的筛选条件。

## 打开分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 使用编辑器顶部的页面选择器打开 **商品系列 → 默认商品系列**，或其他商品系列模板。
4. 在左侧模板列表中点击 **Collection**。
5. 完成设置后，点击右上角的 **保存**。

如果不同商品系列需要不同的标题、说明或页面布局，可以创建多个商品系列模板，并把相应模板分配给指定商品系列。

## Color Scheme

选择当前分区使用的全局配色方案，影响背景、标题、说明、筛选器、排序控件、商品信息和装饰元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用该方案的分区，因此应检查标题、未选中与已选中的筛选项、商品价格和链接与背景之间是否具有足够的对比度。

## Font

### Heading size

设置页面标题的字号。较大的字号能够突出商品系列主题，但会增加标题区域高度；长标题应同时检查桌面端和移动端换行。

### Header

设置商品系列页面的主标题。该字段支持基础富文本格式、链接和动态来源。

可以填写通用标题，也可以连接当前商品系列名称。如果多个商品系列共用同一模板，建议使用动态来源或保持标题具有通用性，避免所有系列显示同一个不相关标题。

### Description

设置主标题下方的介绍文字。该字段支持文字样式、加粗、斜体、链接、项目符号和编号列表，并可连接动态来源。

建议用一至两句话说明系列定位、适用场景或主要特点。多个系列共用模板时，可连接当前商品系列说明，以便每个页面显示对应内容。

## Subtitle

### Subtitle size

设置筛选项、排序入口等次级界面文字的字号。字号过大时，筛选值可能在较窄屏幕上换行或减少同屏显示数量。

## 旋转徽章

### Rotate Badge Text

设置圆形徽章外围显示的文字。该字段支持动态来源，适合展示简短的品牌卖点、选品标准或服务承诺。

### Rotate Badge Percent

设置圆形徽章中心显示的数值或短文本。该字段支持动态来源。

外围文字不宜过长，中心内容建议使用简短数值或标识。徽章仅用于展示信息，不会自动创建折扣、质保或其他业务规则；相关声明必须与实际政策一致。

不需要徽章时，可清空两个字段，并检查页面是否仍保留无内容的装饰区域。

## Products per page

设置每页最多显示的商品数量。当当前商品系列中的商品数量超过该值时，页面会显示分页或主题提供的后续加载方式。

每页商品越多，顾客翻页次数越少，但页面长度、图片请求和初次加载负担也会增加。应结合商品图片大小、网格列数和移动端体验设置合适数量。

该设置不会改变商品系列包含的商品总数，只限制单页显示数量。

## Enable filtering

控制是否允许顾客筛选商品。开启后，页面会显示筛选入口及可用筛选值；关闭后，筛选界面不会显示，但商品系列内容不会被删除。

筛选条件不是在该分区中逐项创建的。点击设置下方的 **Search & Discovery app** 链接，或进入 Shopify **Search & Discovery** 应用，配置库存、价格、商品选项和商品元字段等筛选条件。

如果筛选项没有出现，请检查：

- Search & Discovery 中是否已添加该筛选条件。
- 当前商品系列内是否存在对应值。
- 商品类型、供应商、标签、变体选项或元字段是否填写一致。
- 当前市场中是否有符合条件且可售的商品。

::: tip 筛选命名
只保留能够帮助顾客做决定的筛选条件，并统一颜色、尺寸、材质等属性的写法。例如不要同时使用“深绿色”“墨绿”和“Dark Green”表示同一种颜色。
:::

## Desktop filter layout

设置桌面端筛选器的展示方式。选择 **Drawer** 时，顾客点击筛选入口后，会通过抽屉面板查看和选择条件。

若主题提供其他布局，可根据筛选项数量和页面宽度选择。筛选条件较多时，抽屉通常能为商品网格保留更多横向空间。

## Visible filter values before show more

设置每组筛选条件在出现“显示更多”入口之前，直接展示的筛选值数量。

数值较小可以缩短筛选面板，但常用选项可能被折叠；数值较大会增加面板长度。应根据每组筛选值的实际数量和使用频率调整。

## Enable sorting

控制是否显示商品排序功能。开启后，顾客可以通过排序控件切换推荐、畅销、名称、价格或上架时间等可用顺序。

该开关只决定顾客能否使用排序控件。页面首次打开时的商品顺序，应在 Shopify 后台编辑对应商品系列时设置。

筛选与排序可以同时启用。测试时应确认先筛选再排序以及清除筛选后的商品数量和顺序均正确。

## 商品网格列数

### Desktop product card columns

设置桌面端每行显示的商品卡片数量，可选择 **3 columns** 或 **4 columns**。

三列卡片较宽，适合长标题或信息较多的商品；四列能够同屏展示更多商品，但单张卡片空间更小。

### Mobile product card columns

设置移动端每行显示的商品卡片数量，可选择 **1 column** 或 **2 columns**。

一列更便于展示大图和完整商品信息；两列浏览效率更高，但需要重点检查标题、价格、促销标签和按钮是否拥挤。

商品卡片的图片比例、边框、圆角、阴影等通用样式可在 **主题设置 → Product cards** 中调整。

## Padding

- **Padding Top**：设置分区顶部与页眉或上一个分区之间的留白，单位为 `px`。
- **Padding Bottom**：设置商品列表与下一个分区或页脚之间的留白，单位为 `px`。

可先让上下间距保持一致，再根据标题高度、商品数量和分页区域调整。移动端页面通常较长，不宜使用过大的底部留白。

## 配置示例

| 设置项 | 示例值 |
| --- | --- |
| Color Scheme | 浅色高对比方案 |
| Heading size | `4XL` |
| Header | 连接当前商品系列名称 |
| Description | 连接当前商品系列说明 |
| Subtitle size | `XL` |
| Rotate Badge Text | `严选材质 · 用心设计` |
| Rotate Badge Percent | `100%` |
| Products per page | `12` |
| Enable filtering | 开启 |
| Desktop filter layout | `Drawer` |
| Visible filter values before show more | `6` |
| Enable sorting | 开启 |
| Desktop product card columns | `4 columns` |
| Mobile product card columns | `2 columns` |
| Padding Top / Bottom | `32 px / 32 px` |

## 发布前检查

1. 确认当前模板已分配给正确的商品系列。
2. 检查商品系列及其商品已发布到在线商店，图片、标题、价格和库存信息正确。
3. 如果多个系列共用模板，逐个确认 Header 和 Description 显示对应的动态内容。
4. 检查旋转徽章的文字长度、中心内容、事实依据和移动端位置。
5. 测试每页商品数量不足、等于和超过 **Products per page** 时的显示与分页。
6. 打开筛选器，逐项测试筛选、显示更多、清除条件和结果数量。
7. 测试所有排序选项，并确认默认顺序与 Shopify 后台的商品系列设置一致。
8. 组合使用筛选与排序，确认商品列表和浏览器返回操作保持正常。
9. 分别测试桌面端三列、四列以及移动端一列、两列布局。
10. 检查长商品标题、促销价格、缺货状态和多变体商品的卡片排版。
11. 使用键盘操作筛选、排序、商品链接和分页控件，确认焦点状态清晰可见。
12. 检查标题、说明、筛选状态、排序控件和商品卡片与背景之间的对比度。
13. 点击 **保存**，再从实际店面完成筛选、排序、翻页和进入商品详情页的完整流程。
