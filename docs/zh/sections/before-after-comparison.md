# 前后对比滑块展示（Before After Comparison）

Before After Comparison（前后对比）分区用于将两张图片叠放在同一画面中，并通过可拖动的分隔控件展示变化前后的差异。它适合呈现产品使用效果、空间改造、修复过程、材质处理或设计方案对比。

![Before After Comparison 设置面板与前台效果](/images/zh/before-after-comparison.svg)

## 添加分区

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 打开需要编辑的页面或模板。
4. 点击 **添加分区**，选择 **Before After Comparison**。
5. 分别上传 Before Image 和 After Image。
6. 配置标题、标签、图片比例和间距。
7. 点击右上角的 **保存**。

## Color Scheme

选择当前分区使用的全局配色方案，影响背景、标题、副标题、图片标签和对比控件等元素。

点击 **编辑方案** 可进入全局颜色设置。修改方案本身会同时影响其他使用同一方案的分区，应确保文字、标签和滑块控件在浅色与深色图片上都清晰可见。

## Font

面板中的 **Font** 为字体相关设置区域。若当前主题版本未显示独立选项，分区会沿用主题的全局字体设置；可在 **主题设置 → Typography** 中调整全站标题和正文字体。

## Heading

### Heading size

设置分区主标题的字号，例如 **2XL**。标题较长时应检查桌面端和移动端的换行，并避免标题占用过多首屏空间。

### Heading

设置分区主标题，例如 **Up To 50% Off Everything**。该字段支持基础富文本格式、链接和动态来源。

标题应简要说明对比主题或核心变化。不需要主标题时可以留空。

## Subtitle

### Subtitle size

设置辅助标题的字号，例如 **3XL**。应根据 Heading 与 Sub Heading 的实际内容建立清晰层级，不要只依据选项名称判断视觉重要性。

### Sub Heading

设置辅助标题，例如 **Lorem Ipsum**。该字段支持基础富文本格式、链接和动态来源。

辅助标题可以用于展示活动类型、产品名称、项目阶段或简短说明。不需要时可以留空。

## Image ratio

设置前后对比图片容器的宽高比例。选择 **Wide (16:9)** 时，容器采用宽屏比例，适合横向场景、室内空间和宽幅产品图片。

固定比例可能裁切图片边缘。上传前应确保两张图片具有相同比例，并将关键主体放在相同位置；修改比例后要重新检查桌面端和移动端裁切。

## Before Image

选择变化前的图片，也可以通过动态来源连接符合条件的图片字段。

Before Image 应准确代表起始状态。建议使用清晰原图，不要预先添加文字、标签或边框，以免与分区自带标签和滑块控件重叠。

## After Image

选择变化后的图片，也可以通过动态来源连接符合条件的图片字段。

After Image 应与 Before Image 使用相同的拍摄角度、构图、尺寸和主体位置。对比条件越一致，顾客越容易判断实际变化。

::: tip 图片准备
拍摄前后图片时，尽量固定相机位置、焦距、光线和背景。若两张图片的视角差异过大，拖动滑块时会产生跳动感，也可能削弱对比结果的可信度。
:::

## Before Label

设置 Before Image 的标签文字，例如 `BEFORE`。该字段支持动态来源。

标签应简短明确，并与店铺当前语言保持一致。中文店铺可使用“使用前”“改造前”或“处理前”等更具体的文字。

## After Label

设置 After Image 的标签文字，例如 `AFTER`。该字段支持动态来源。

Before Label 和 After Label 应采用成对且含义一致的表达，例如“改造前 / 改造后”，避免使用无法对应的两个概念。

## Padding

- **Padding Top**：设置分区顶部留白，单位为 `px`，例如 `32 px`。
- **Padding Bottom**：设置分区底部留白，单位为 `px`，例如 `32 px`。

顶部间距影响标题与上一个分区的距离，底部间距影响对比图片与后续内容的衔接。移动端标题换行较多时，可能需要重新调整上下留白。

## 前台交互

两张图片共用同一个显示区域，分隔控件决定各自可见的范围。顾客拖动控件时，应能连续查看从 Before 到 After 的变化。

为了保证交互清晰：

- 滑块手柄应与图片保持足够对比度。
- Before Label 和 After Label 不应被滑块或图片主体遮挡。
- 拖动时图片不应发生缩放、位移或明显跳动。
- 触摸设备上的手柄应具有足够大的操作区域。
- 页面纵向滚动不应与滑块横向拖动产生严重冲突。

## 内容与合规建议

- 使用真实、可验证的前后效果，避免通过不同光线、角度或后期处理夸大变化。
- 美容、健康或性能类对比应符合销售地区的广告要求，并在需要时添加结果差异说明。
- 不要在未经授权的情况下使用顾客照片、个人信息或受版权保护的图片。
- 图片替代文本应分别描述前后状态，不能只写“Before”和“After”。

## 参考配置

| 设置项 | 示例值 |

| Padding Top / Bottom | `32 px / 32 px` |

## 发布前检查

1. 确认 Before Image 和 After Image 顺序正确，没有上传反向。
2. 检查两张图片的尺寸、比例、拍摄角度和主体位置是否一致。
3. 从最左侧拖到最右侧，确认整个对比过程连续且没有空白区域。
4. 在桌面端使用鼠标，在移动端使用触控手势测试滑块。
5. 使用键盘 `Tab`、方向键和 `Enter` 检查对比控件是否可操作，并确认焦点状态清晰。
6. 检查标题、辅助标题、标签和滑块在两张图片上都具有足够对比度。
7. 确认宽屏比例下没有裁掉关键主体，并测试其他可用图片比例。
8. 检查长标题、隐藏标题和不同语言标签的排版。
9. 点击 **保存**，并在实际店面分别完成桌面端和移动端测试。
