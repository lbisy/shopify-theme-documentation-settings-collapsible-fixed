# 技术支持与主题更新

本页介绍 Esen 主题的技术支持范围、问题反馈方式、更新前的准备工作以及版本记录。

![Esen 技术支持与主题更新](/images/zh/support.svg)

## 获取技术支持

如需帮助，请通过购买或下载 Esen 主题的平台所提供的官方支持渠道联系我们。为便于快速定位问题，请尽可能一次性提供以下信息：

- Shopify 商店地址；
- 当前使用的 Esen 主题版本；
- 出现问题的页面或模板链接；
- 问题的具体表现以及预期结果；
- 可以稳定复现问题的操作步骤；
- 使用的设备、操作系统和浏览器版本；
- 相关截图或屏幕录制；
- 问题出现前安装的应用、启用的应用嵌入或添加的自定义代码。

::: warning 账户与数据安全
请勿发送 Shopify 管理员密码、客户个人信息、付款信息、API 密钥或访问令牌。如果排查问题确实需要协作者权限，请仅通过安全的官方渠道提供最小必要权限，并在问题解决后及时撤销。
:::

## 提交问题前的自查

在提交技术支持请求前，建议先完成以下检查：

1. 确认问题发生在当前使用的 Esen 主题副本及正确的页面模板中。
2. 保存主题编辑器中的更改，并使用浏览器无痕窗口重新测试。
3. 分别在桌面端和移动端检查问题是否存在。
4. 复制一份未发布的主题，在副本中暂时停用最近添加的应用嵌入或自定义代码。
5. 检查相关商品、页面、菜单和内容是否已发布到正确的销售渠道。
6. 查看[常见问题](/zh/faq)，确认是否已有对应的解决方法。

## 标准支持范围

Esen 的标准技术支持通常包括：

- 主题安装与基础设置指导；
- 主题编辑器中 Esen 内置功能的使用说明；
- 文档中功能或设置项的解释；
- 可复现的主题功能异常与兼容性问题排查；
- 已确认主题问题的临时解决方案或修复说明；
- Esen 新版本的安装与常规更新指导。

## 通常不在标准支持范围

以下项目通常不属于 Esen 的标准技术支持范围：

- 第三方应用的安装、设置、故障或兼容性开发；
- 定制页面、全新功能或特定视觉效果的开发；
- 自定义 Liquid、CSS 或 JavaScript 的编写与调试；
- 因直接修改主题源代码而产生的问题；
- Shopify 账户、套餐、付款、物流、税费或域名相关问题；
- 商店内容录入、商品数据整理或日常运营工作。

对于此类问题，我们可以帮助判断问题来源，并建议联系相应的应用开发者、Shopify 支持或专业开发人员。

## 安装主题更新前

Esen 主题更新可能包含错误修复、功能改进和兼容性调整。安装未来版本前，请先完成以下准备：

1. 在 Shopify 后台复制当前已发布的主题，保留可随时恢复的备份。
2. 记录当前主题版本，以及已安装的应用和自定义修改。
3. 备份自定义 Liquid、CSS、JavaScript 和其他手动修改过的文件。
4. 将新版本作为未发布主题安装，不要直接替换线上主题。
5. 根据需要重新配置主题设置、模板、应用嵌入和自定义代码。
6. 完成关键流程测试后再发布新版本，并暂时保留旧版本以便回退。

::: warning 更新不会自动合并自定义内容
新安装的主题版本是一个独立主题副本。旧版本中的主题设置、自定义模板、应用配置及代码修改不一定会自动迁移，请在发布前逐项核对。
:::

## 更新后的检查清单

发布更新前，建议至少检查以下内容：

- 首页及主要营销页面；
- 商品详情、规格选择和加入购物车；
- 购物车、搜索和筛选功能；
- 联系表单、订阅表单及其他交互组件；
- 桌面端、平板和移动端显示效果；
- 多语言、市场和货币设置；
- 第三方应用区块与应用嵌入；
- 自定义代码和像素追踪是否正常运行。

## 更新与兼容性说明

- 主题更新不会同时更新 Shopify 平台或第三方应用。
- 自定义代码及第三方应用的兼容性需要单独测试。
- 安装新版本前，请先阅读对应版本的更新说明。
- 在新版本完成验证前，请勿删除或覆盖当前线上主题。

## 版本记录

### Esen 1.0.0

**首次正式发布**

- 基于 Shopify Online Store 2.0 架构；
- 支持桌面端、平板和移动端响应式显示；
- 提供颜色、排版、布局、按钮、输入框和动画等全局主题设置；
- 包含商品、商品系列、购物车、搜索、博客、文章及内容页面模板；

:::
