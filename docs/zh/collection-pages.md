# 商品系列页面

商品系列页面用于集中展示同一系列中的商品，并帮助顾客通过筛选和排序快速缩小选择范围。请先在 Shopify 后台创建商品系列、添加商品并完善商品信息，再在主题编辑器中配置标题、说明、商品数量、筛选、排序、网格列数和页面间距。

![商品系列页面设置](/images/zh/collection-page.svg)

## 创建商品系列并添加商品

1. 进入 **Shopify 后台 → 产品 → 商品系列**。
2. 点击 **创建商品系列**，填写标题和说明。
3. 选择商品系列类型：
   - **手动**：逐个选择要加入该系列的商品。
   - **智能**：设置条件，由 Shopify 自动加入符合条件的商品。
4. 按需要上传商品系列图片，并设置默认商品排序方式。
5. 完善搜索引擎列表信息，将商品系列设置为对 **在线商店** 销售渠道可见，然后保存。
6. 前往 **内容 → 菜单**（部分后台版本显示为 **在线商店 → 导航**），将商品系列链接添加到页眉或其他菜单。

同时确认系列内商品已发布到在线商店，并具有正确的图片、价格、库存、商品类型、供应商、标签及变体信息。这些数据会直接影响商品卡片、库存筛选和其他筛选条件的显示。

## 打开商品系列模板

1. 进入 **在线商店 → 主题**，在当前主题上点击 **自定义**。
2. 使用编辑器顶部的页面选择器打开 **商品系列 → 默认商品系列**。
3. 在左侧模板列表中点击 **Collection** 分区。
4. 在右侧完成设置后点击 **保存**。

如果不同商品系列需要不同的版式或文案，可以新建商品系列模板，将模板分配给指定商品系列后再单独配置。

## Color Scheme

**Color Scheme** 用于设置 Collection 分区的背景、文字、筛选和排序控件、商品区域及装饰元素的配色。点击 **编辑方案** 可以进入全局颜色设置。

::: warning 注意
修改全局配色方案会同时影响其他使用该方案的分区。保存前请检查标题、说明、筛选状态、商品信息和背景之间是否具有足够的对比度。
:::

## Font

- **Heading size**：设置商品系列主标题字号。
- **Header**：设置页面主标题，例如 `Products`。该字段支持加粗、斜体和链接等基础格式；留空时的具体显示效果取决于主题实现。
- **Description**：设置标题下方的介绍文字，支持基础富文本和列表格式。

标题应简短并能概括当前系列，说明建议控制在一至两句，并使用与商店语言一致的文案。

## Subtitle

- **Subtitle size**：设置筛选分类、排序等次级控制项的字号。

字号过大时，分类项在较窄屏幕上可能换行或减少同屏显示数量。修改后请同时预览桌面端和移动端。

## 旋转徽章

- **Rotate Badge Text**：设置圆形旋转徽章外围的文字。
- **Rotate Badge Percent**：设置徽章中心内容。

旋转徽章适合展示简短的品牌卖点或承诺。外围文字不宜过长，中心内容建议使用百分比或短词；修改后请检查文字是否完整环绕，且不会与标题、说明或商品控件重叠。

## 每页商品数量

**Products per page** 设置每页最多显示的商品数量。当商品系列中的商品数量超过该值时，页面会显示分页控件。

每页商品越多，顾客翻页次数越少，但首屏加载内容和页面长度也会增加。设置时应结合图片大小、移动端体验和商品总量综合考虑。

## 筛选

- **Enable filtering**：开启或关闭商品筛选。启用后，顾客可以按已配置的条件缩小商品范围。
- **Desktop filter layout**：设置桌面端筛选器的展示方式。选择 **Drawer** 后，顾客点击 **Filter**，筛选项会从页面侧边以抽屉形式打开。
- **Visible filter values before show more**：设置每组筛选条件在出现“显示更多”之前直接展示的选项数量。

筛选抽屉可展示 **Availability**（库存状态）、**Price**（价格范围）及其他已配置的筛选条件。选择条件后，顾客可点击底部的 **View Result** 查看结果；按钮旁的数字表示当前符合条件的商品数量。

筛选项不是在本分区中逐项创建的。请点击设置中的 **Search & Discovery app** 链接，或前往 Shopify 后台的 **Search & Discovery** 应用配置可用筛选条件。筛选结果依赖商品数据，因此建议先统一整理商品类型、供应商、标签、变体选项和相关元字段。

::: tip 建议
只保留真正有助于顾客决策的筛选项。选项过多或命名不一致会增加查找成本；价格、库存、颜色、尺寸等常用条件应使用清晰且统一的数据。
:::

## 排序

**Enable sorting** 控制是否显示排序功能。开启后，顾客可以通过页面中的 **Sort by** 控件更改商品顺序。可用选项通常以 Shopify 的商品系列排序能力为基础，例如推荐、畅销、名称、价格和上架时间。

主题编辑器中的开关只控制顾客是否可以使用排序控件；商品系列的初始排序方式请在 Shopify 后台编辑该商品系列时设置。

## 商品网格列数

- **Desktop product card columns**：设置桌面端每行的商品卡片数量，可选择 **3 columns** 或 **4 columns**。
- **Mobile product card columns**：设置移动端每行的商品卡片数量，可选择 **1 column** 或 **2 columns**。

列数越多，单张商品卡片越窄。选择四列或移动端两列时，应重点检查商品标题、价格、促销信息和按钮是否拥挤或异常换行。商品图片比例、卡片边框、圆角和阴影等通用样式可在主题设置的 **Product cards（商品卡片）** 中调整。

## Padding

- **Padding Top**：调整 Collection 分区顶部留白，单位为 `px`。
- **Padding Bottom**：调整 Collection 分区底部留白，单位为 `px`。

如果页面标题离页眉过近，可增加顶部间距；如果商品网格与后续分区距离过大，可减小底部间距。

## 发布前检查

- 确认商品系列已发布到在线商店，系列内商品、图片、价格和库存状态正确。
- 检查页面标题、说明和旋转徽章在桌面端与移动端均无重叠或异常换行。
- 打开筛选抽屉，逐项测试库存、价格及其他筛选条件，并确认结果数量正确。
- 测试清除筛选、关闭抽屉和 **View Result** 按钮，确认页面状态更新正常。
- 测试各个排序选项，确认商品顺序符合预期。
- 当商品数量超过单页上限时，检查分页控件和翻页后的商品列表。
- 分别预览桌面端三列、四列以及移动端一列、两列布局，检查图片裁切、文字换行和触控区域。
- 确认菜单中的商品系列链接可以打开正确页面，并保存主题更改。
