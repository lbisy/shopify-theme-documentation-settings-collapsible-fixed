# 页眉、公告栏与页脚

页眉（Header）、公告栏（Announcement Bar）和页脚（Footer）属于全站共用区域。修改后通常会同步应用到使用同一页眉组或页脚组的页面，保存前请同时预览桌面端和移动端效果。

## 开始配置

1. 进入 **Shopify 后台 → 在线商店 → 主题**。
2. 在当前主题上点击 **自定义**。
3. 在编辑器左侧找到 **Header** 或 **Footer** 分组。
4. 点击需要配置的 **Announcement Bar**、**Header** 或 **Footer**。
5. 完成设置后点击右上角 **保存**。

在配置导航前，请先前往 **Shopify 后台 → 内容 → 菜单**（部分后台版本显示为 **在线商店 → 导航**）创建菜单，再将其分配给 Header 或 Footer。Logo 需要在主题设置中上传或修改。

## 页眉（Header）

页眉用于展示 Logo、主导航以及搜索、账户、购物袋等入口。

![页眉设置](/images/zh/header-settings.svg)

### Header

- **Logo Position**：设置 Logo 在页眉中的位置。
- **Edit your logo in theme settings**：点击链接进入主题设置，上传或替换店铺 Logo。若未上传 Logo，主题通常会显示店铺名称。

### Color

- **Color Scheme**：设置页眉背景、文字和图标使用的配色方案。
- **Menu Color Scheme**：设置桌面端下拉菜单或超级菜单的配色方案。
- **Mobile Menu Color Scheme**：设置移动端抽屉菜单的配色方案。
- **编辑方案**：进入全局颜色设置，调整所选方案的背景色、文字色和强调色。修改配色方案可能同时影响其他使用该方案的区域。

建议确保文字、图标与背景之间有足够对比度，并分别检查普通页眉、展开后的菜单和移动端菜单。

### Menu

- **Menu**：选择页眉使用的导航菜单，例如 **Main menu**。菜单层级和链接内容需要在 Shopify 菜单管理中编辑。

### Menu Type

- **Menu Type**：选择导航菜单的展示类型，例如 **Super Menu**。
- **Desktop menu trigger**：设置桌面端打开子菜单的方式。
  - **Hover**：鼠标悬停时展开，浏览更快捷。
  - **Click**：点击后展开，可减少误触，也更适合需要明确操作的菜单。
- **Super Menu Watermark**：设置超级菜单中的品牌水印文字，例如品牌名称或商标文字。
- **Enable Product Series Tabs**：开启后，在超级菜单中显示商品系列标签页。
- **Series Collection 1 / 2 / 3**：为商品系列标签页分别选择集合。此设置需要先开启 **Enable Product Series Tabs** 才会生效；未使用的系列可留空。

配置超级菜单时，建议主菜单层级保持清晰，系列名称尽量简短，并确认每个菜单项和集合均已发布到当前销售渠道。

## 公告栏（Announcement Bar）

公告栏位于页眉上方，适合展示配送提示、促销活动、营业通知或重要政策入口。

1. 在编辑器左侧展开 **Announcement Bar**。
2. 添加或选择公告内容块。
3. 填写简短文案，并在需要时配置跳转链接。
4. 检查文字与背景对比度以及移动端换行效果。

公告内容建议只保留一项核心信息；若使用多条公告，请将最重要的内容排在前面，并定期移除已过期活动。

## 页脚（Footer）

页脚用于承载品牌信息、引导内容、商店关注入口以及底部导航等辅助信息。

![页脚设置](/images/zh/footer-settings.svg)

### 基础外观

- **Color Scheme**：选择页脚使用的全局配色方案。
- **Background Image**：上传页脚背景图片。建议使用适合横向铺展的图片，并确认文字在图片上仍清晰可读。

### Brand

- **Brand Name**：填写品牌名称，支持富文本格式，例如 `Eccenova®`。
- **Brush Texture Image**：上传品牌区域使用的笔刷或纹理装饰图。建议使用透明背景的 PNG 或 WebP 图片，避免遮挡文字。
- **Location Text**：填写品牌或门店所在地，例如 `Shibuya, JP`。

### Right Content

- **Heading**：设置右侧内容区标题，例如品牌卖点或推荐内容标题。
- **Description**：填写标题下方的说明文字，支持基础富文本格式和列表。
- **Link**：选择按钮或文字链接需要跳转的集合、商品、页面或网址。
- **Link Text**：设置链接在前台显示的文字。若不需要显示入口，可将链接和链接文字同时留空。

### Follow on Shop

- **Enable Follow on Shop button**：控制是否显示 **Follow on Shop** 按钮。开启前请确认商店已满足 Shop 渠道的相关配置要求，并在预览中测试按钮是否正常显示。

## 发布前检查

- 确认 Logo、页眉和页脚配色符合品牌规范，文字与背景对比清晰。
- 检查主菜单、下拉菜单、超级菜单以及页脚中的全部链接。
- 分别测试 **Hover** 或 **Click** 触发方式是否符合预期。
- 若启用商品系列标签页，确认所选集合已发布且包含正确商品。
- 检查背景图和纹理图在不同屏幕宽度下的裁切效果。
- 在桌面端和移动端确认菜单展开、文字换行、图标位置和页脚排列。
- 确认公告内容未过期，最后点击 **保存**。
