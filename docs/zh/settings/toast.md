# Toast

Toast（轻提示）是在页面上短暂显示的状态消息，例如操作成功、警告或错误提醒。本设置用于统一调整提示框的位置、停留时间和外观。

## 打开设置

1. 进入 Shopify 后台的 **在线商店 > 主题**。
2. 找到当前主题，点击 **自定义**。
3. 打开 **主题设置 > Toast**。

![Toast 设置](/images/zh/settings/toast.svg)

## 基础设置

### Toast radius

设置提示框的圆角大小。数值越大，四角越圆润；数值越小，外观越方正。

### Position

设置提示框在页面中的显示位置。例如 **Top center** 表示显示在页面顶部中央。

选择位置时，应避免遮挡页眉导航、购物车按钮或移动端的重要操作区域。

### Auto-dismiss (seconds)

设置提示框自动关闭前的停留时间，单位为秒。

- 时间过短，用户可能来不及阅读。
- 时间过长，提示可能遮挡页面内容。
- 内容较长或较重要时，可以适当增加停留时间。

## Preview

**Preview** 用于在主题编辑器中预览不同类型的 Toast。选择 **Off** 时关闭预览。

开启 **Keep preview visible until dismissed** 后，预览提示会一直显示，直到手动关闭，方便检查文字、颜色、位置和阴影。

::: info 提示
Preview 仅用于编辑时检查外观，不会让前台提示一直显示。前台的实际停留时间由 **Auto-dismiss** 控制。
:::

## Shadow

阴影用于增强 Toast 与页面背景之间的层次。

| 设置项 | 作用 |
| --- | --- |
| **Toast shadow opacity** | 设置阴影透明度；`0%` 表示不显示阴影。 |
| **Toast shadow horizontal offset** | 设置阴影左右偏移；正数向右，负数向左。 |
| **Toast shadow vertical offset** | 设置阴影上下偏移；正数向下，负数向上。 |
| **Toast shadow blur** | 设置阴影模糊程度；数值越大，阴影越柔和。 |

建议使用较低透明度、轻微向下偏移和适度模糊，避免阴影过重。

## 颜色

Toast 的成功、警告、错误和信息颜色来自 **主题设置 > Colors > Feedback messages**。如需修改提示框的背景或文字颜色，请编辑对应配色方案。

## 保存与检查

保存设置前，使用 Preview 检查：

- 提示框没有遮挡重要内容。
- 文字在背景上清晰可读。
- 自动关闭时间足够阅读。
- 阴影和圆角与主题风格一致。
- 桌面端和移动端均能完整显示。

完成后点击主题编辑器右上角的 **保存**。
