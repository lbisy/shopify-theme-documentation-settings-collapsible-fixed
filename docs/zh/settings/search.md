# Search

Search（搜索）设置用于控制商店是否启用预测搜索。预测搜索可以在顾客输入关键词的过程中显示匹配建议，帮助顾客更快找到商品和相关内容。

## 打开 Search 设置

1. 登录 Shopify 后台。
2. 进入 **在线商店 > 主题**。
3. 找到当前使用的主题，点击 **自定义**。
4. 点击左侧的 **主题设置**。
5. 选择 **Search**。

![Search 设置](/images/zh/settings/search.svg)

## Enable predictive search

**Enable predictive search** 用于开启或关闭预测搜索功能。

### 开启预测搜索

开启后，顾客在搜索框中输入关键词时，系统会在输入完成前显示匹配建议。根据主题支持的结果类型和商店内容，建议中可能包含商品或其他相关搜索结果。

开启预测搜索的优点包括：

- 减少顾客输入完整关键词所需的时间。
- 帮助顾客确认商品名称或关键词拼写。
- 让顾客在进入完整搜索结果页之前快速打开匹配内容。
- 对商品数量较多的商店更方便。

::: tip 适用场景
如果商店包含较多商品、商品名称明确，或者顾客经常通过搜索寻找特定商品，建议开启预测搜索。
:::

### 关闭预测搜索

关闭后，顾客输入关键词时不会显示即时建议。顾客需要提交搜索，随后在完整搜索结果页面中查看匹配内容。

关闭预测搜索适合以下情况：

- 商店商品和内容数量较少。
- 希望搜索界面保持简单、减少动态内容。
- 预测建议与商店实际搜索需求不匹配。
- 希望顾客集中在完整结果页面中筛选内容。

关闭此功能不会移除商店搜索，普通搜索结果页面仍然可以使用。

## 预测结果的影响因素

预测搜索显示的内容会受到商店数据和顾客输入内容影响。为了提高搜索质量，建议：

- 使用清晰、准确的商品标题。
- 在商品信息中使用顾客常用的关键词。
- 避免多个商品使用难以区分的相同名称。
- 确认需要展示的商品已经发布到在线商店销售渠道。
- 检查商品图片、价格和可售状态是否正确。

::: info 提示
如果刚刚修改商品标题、商品状态或搜索相关数据，前台结果可能需要短暂时间才能更新。可稍后刷新页面再次测试。
:::

## 保存与检查

完成设置后，点击主题编辑器右上角的 **保存**，然后在商店预览中测试搜索。

建议检查以下内容：

1. 点击页眉中的搜索按钮或搜索框。
2. 输入完整商品名称的一部分。
3. 输入常见分类词、品牌词或商品特征。
4. 检查预测结果是否与关键词相关。
5. 点击预测结果，确认能够进入正确页面。
6. 提交搜索，确认完整搜索结果页可以正常打开。
7. 输入没有匹配结果的关键词，检查空结果提示是否清晰。

还应分别检查桌面端和移动端：

- 搜索建议是否完整显示在屏幕内。
- 较长的商品名称是否正常换行或截断。
- 商品图片、标题和价格是否排列清晰。
- 使用键盘方向键、`Tab` 和 `Enter` 时是否能够操作搜索建议。
- 关闭搜索或返回页面后，页面是否能够正常继续浏览。

::: tip 建议
测试时不要只使用一个商品名称。建议选择热门商品、名称相似的商品以及不同语言的关键词，确认预测结果在真实使用场景中仍然准确、易用。
:::
