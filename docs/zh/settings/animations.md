# Animations

Animations（动画）设置用于控制页面内容和媒体元素进入视口时的展示效果。适当的动画可以增强页面层次感，引导用户浏览内容；过多或过快的动画则可能分散注意力。

## 打开 Animations 设置

1. 登录 Shopify 后台。
2. 进入 **在线商店 > 主题**。
3. 找到当前使用的主题，点击 **自定义**。
4. 点击左侧的 **主题设置**。
5. 选择 **Animations**。

![Animations 设置](/images/zh/settings/animations.svg)

## Enable motion

**Enable motion** 是全站动画的总开关。

- 开启后，页面会按照下方设置显示内容和媒体出现动画。
- 关闭后，页面元素会直接显示，不再播放出现动画。

如果商店页面包含较多图片、视频或复杂分区，建议分别在桌面端和移动端预览动画效果，确认页面浏览依然流畅。

::: tip 提示
关闭动画不会隐藏页面内容，只会让内容立即显示。
:::

## Content reveal style

**Content reveal style** 用于设置标题、正文、按钮、商品信息等内容进入视口时的出现方式。

- **None**：不使用出现动画，内容立即显示。
- **Fade**：内容通过透明度变化逐渐显示，效果柔和、干扰较少。
- **Rise**：内容在淡入的同时轻微向上移动，能够增强页面的层次感和浏览节奏。

对于文字较多的页面，建议使用 **Fade**；如果希望首页或品牌展示区域更有动态感，可以选择 **Rise**。

## Media reveal style

**Media reveal style** 用于控制图片、视频和其他媒体元素进入视口时的出现方式。

- **None**：媒体立即显示，不播放出现动画。
- **Fade**：媒体通过透明度变化逐渐显示。
- **Zoom**：媒体在淡入的同时进行轻微缩放，视觉效果更加突出。

::: warning 注意
同一页面中使用大量 Zoom 动画可能会分散用户注意力。建议重点检查商品图片、轮播图和大幅横幅，避免动画效果过于频繁。
:::

## Motion speed

**Motion speed** 用于设置内容和媒体出现动画的整体播放速度。

- **Slow**：动画持续时间较长，效果更舒缓，适合强调氛围感的页面。
- **Normal**：在视觉效果和浏览效率之间保持平衡，适合大多数商店。
- **Fast**：动画更快完成，页面响应感更强，适合内容密度较高的页面。

动画速度会同时影响内容和媒体的出现效果。选择速度后，建议滚动预览多个页面分区，检查动画节奏是否自然。

## Reveal behavior

**Reveal behavior** 用于决定元素的出现动画播放一次，还是在元素重新进入视口时再次播放。

- **Once**：向下滚动页面时，每个元素只播放一次出现动画。之后再次进入视口时不会重复播放。
- **Always**：元素每次重新进入视口时都会再次播放出现动画。

对于商品列表、博客文章和内容较长的页面，建议使用 **Once**，减少重复动画对阅读的干扰。如果页面内容较少，并希望每次滚动返回时都保留动态效果，可以选择 **Always**。

::: info 减少动态效果
当访客的设备启用了“减少动态效果”偏好，或者主题的动画总开关被关闭时，页面内容会立即显示，不会等待动画完成。这有助于为容易受到动态效果影响的用户提供更舒适的浏览体验。
:::

## 推荐组合

可以根据商店的视觉风格选择以下组合：

- **简洁快速**：Content 选择 **Fade**，Media 选择 **Fade**，速度选择 **Fast**，行为选择 **Once**。
- **自然平衡**：Content 选择 **Rise**，Media 选择 **Fade**，速度选择 **Normal**，行为选择 **Once**。
- **氛围展示**：Content 选择 **Rise**，Media 选择 **Zoom**，速度选择 **Slow**，行为选择 **Once**。
- **无动画**：关闭 **Enable motion**，或将 Content 和 Media 都设置为 **None**。

## 保存与检查

完成设置后，点击主题编辑器右上角的 **保存**。建议检查以下内容：

- 首页首屏内容是否能够及时显示。
- 向下滚动时，文字和图片的出现节奏是否协调。
- 商品列表中多个卡片连续出现时是否流畅。
- 移动端滚动时是否存在卡顿或明显延迟。
- 选择 Always 后，重复动画是否会干扰阅读。
- 关闭动画或启用减少动态效果后，所有内容是否立即可见。

::: tip 建议
动画应当辅助内容，而不是成为用户浏览页面的阻碍。完成设置后，建议以普通访客的速度完整浏览一次首页、商品列表页和商品详情页。
:::
