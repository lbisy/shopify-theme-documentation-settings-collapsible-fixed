# Layout

Layout（布局）设置用于调整页面内容的最大宽度和小圆角尺寸，从而控制网站整体的空间感与视觉风格。

## 设置页面宽度

**Set Page Width** 用于设置页面内容区域的最大宽度，可设置范围为 **1000px–1600px**。

拖动设置中的滑块即可调整页面宽度。页面宽度不会改变浏览器窗口的大小，而是决定内容在宽屏设备上最多可以占用的横向空间。

- **较窄的宽度**：内容更集中，阅读视线移动距离较短，适合文字较多或希望版面更紧凑的网站。
- **中等宽度**：兼顾内容密度与留白，适合大多数商店。
- **较宽的宽度**：能够展示更多横向内容，适合图片较多、商品卡片较多或希望营造开阔感的网站。

::: tip 建议
调整页面宽度后，请同时检查桌面端的首页、商品列表页和商品详情页，确认文字、图片与卡片之间的比例协调。
:::

## 设置小圆角

**Setting Small Border Radius** 用于设置网站中小圆角元素的圆角大小，共有 **small（小）**、**medium（中）** 和 **large（大）** 三种选项。

- **small**：圆角较小，整体风格更利落、规整。
- **medium**：圆角适中，在简洁感与柔和感之间取得平衡。
- **large**：圆角更明显，整体风格更加柔和、亲切，适合宠物用品、儿童用品等网站。

::: warning 注意
此设置不会影响完全圆角的元素，例如部分按钮和搜索框。
:::

选择圆角大小时，建议结合品牌风格以及图片、卡片和输入框的设计统一考虑。修改后请检查不同页面中的相关元素，确保圆角效果保持一致。
