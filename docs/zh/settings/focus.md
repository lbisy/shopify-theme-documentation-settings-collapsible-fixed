# Focus

Focus（焦点）设置用于调整键盘导航时显示的焦点环。焦点环能够标示当前选中的链接、按钮、输入框或其他可交互元素，帮助用户了解按下 Enter 或 Space 后将操作哪个元素。

## 打开 Focus 设置

1. 登录 Shopify 后台。
2. 进入 **在线商店 > 主题**。
3. 找到当前使用的主题，点击 **自定义**。
4. 点击左侧的 **主题设置**。
5. 选择 **Focus**。

![Focus 设置](/images/zh/settings/focus.svg)

## Focus ring width

**Focus ring width** 用于设置焦点环线条的粗细。

- 数值越大，焦点环越粗，更容易被注意到。
- 数值越小，焦点环越细，视觉效果更加轻巧。
- 如果线条过细，在复杂背景或高分辨率屏幕上可能难以识别。

焦点环需要在浅色和深色背景上都保持清晰。调整宽度时，请同时检查按钮、文字链接、图片链接、输入框和选择器等不同元素。

::: warning 无障碍要求
不建议将焦点环宽度设置为 `0px`。隐藏焦点环会让只使用键盘的访客难以判断当前操作位置。
:::

## Focus ring offset

**Focus ring offset** 用于设置焦点环与元素边缘之间的距离。

- 较小的数值让焦点环贴近元素边缘，整体效果更紧凑。
- 较大的数值会增加焦点环与元素之间的留白，使焦点状态更加突出。
- 设置为 `0px` 时，焦点环紧贴元素轮廓。

适当的偏移可以避免焦点环与按钮边框、输入框边框或链接下划线重叠。如果偏移过大，焦点环可能与附近的文字或其他元素发生视觉冲突。

::: tip 建议
使用带边框的按钮和输入框时，可以保留少量偏移，让焦点环与元素原有边框容易区分。
:::

## Focus ring color

焦点环颜色不在此面板中设置。请进入 **主题设置 > Colors**，打开对应的配色方案，然后调整 **Focus ring color**。

焦点环颜色应当：

- 与当前背景具有明显对比。
- 与元素本身的边框容易区分。
- 在商店使用的不同配色方案中保持可见。
- 尽量保持统一，使用户能够快速识别焦点状态。

::: info 提示
焦点环的宽度和偏移属于全局外观设置，颜色则由各个配色方案分别控制。因此修改后需要检查浅色、深色和强调色方案。
:::

## 如何测试键盘焦点

完成设置后，可以直接使用键盘测试：

1. 打开商店预览页面。
2. 点击浏览器页面的空白位置。
3. 连续按下 `Tab` 键，在链接、按钮和表单字段之间向前移动。
4. 使用 `Shift + Tab` 向后移动。
5. 确认每个可交互元素获得焦点时都有清晰、完整的焦点环。
6. 按下 `Enter` 或 `Space`，确认当前元素能够正常操作。

测试时应重点检查：

- 页眉导航、语言选择和搜索按钮。
- 商品卡片、商品选项和加入购物车按钮。
- 购物车抽屉或购物车页面。
- 联系表单和邮件订阅输入框。
- 弹窗中的按钮、输入框和关闭按钮。
- 页脚链接和社交媒体图标。

## 保存与检查

完成设置后，点击主题编辑器右上角的 **保存**，并分别在桌面端与移动端尺寸下预览。

::: tip 推荐原则
焦点环首先需要清晰可见，其次才是与品牌视觉保持一致。不要仅因为觉得焦点轮廓影响美观而将其隐藏。
:::
