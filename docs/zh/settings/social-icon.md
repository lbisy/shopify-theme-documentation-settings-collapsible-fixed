# Social icon

Social icon（社交媒体图标）用于填写商店的社交平台主页链接。设置后，对应图标会显示在主题支持的位置，例如页脚或社交媒体区域。

## 打开设置

1. 进入 Shopify 后台的 **在线商店 > 主题**。
2. 找到当前主题，点击 **自定义**。
3. 打开 **主题设置 > Social icon**。

![Social icon 设置](/images/zh/settings/social-icon.svg)

## 填写社交平台链接

在对应输入框中粘贴完整的个人主页或品牌主页 URL：

| 设置项 | 链接示例 |
| --- | --- |
| Facebook | `https://www.facebook.com/yourbrand` |
| Instagram | `https://www.instagram.com/yourbrand` |
| YouTube | `https://www.youtube.com/@yourbrand` |
| TikTok | `https://www.tiktok.com/@yourbrand` |
| X / Twitter | `https://x.com/yourbrand` |
| Snapchat | `https://www.snapchat.com/add/yourbrand` |
| Pinterest | `https://www.pinterest.com/yourbrand` |
| Tumblr | `https://yourbrand.tumblr.com` |
| Vimeo | `https://vimeo.com/yourbrand` |

::: warning 注意
请填写以 `https://` 开头的完整网址。不要只填写用户名或 `1`、`2`、`3` 等内容，否则图标可能会打开错误页面。
:::

不需要展示的平台保持为空即可。填写完成后点击右上角的 **保存**。

## 检查结果

保存后，在商店前台确认：

- 已填写的平台显示了正确图标。
- 点击图标能够打开正确的社交主页。
- 未填写的平台没有显示。
- 桌面端和移动端的图标排列正常。

::: tip 建议
尽量使用品牌官方主页链接，并定期检查链接是否仍然有效。
:::
