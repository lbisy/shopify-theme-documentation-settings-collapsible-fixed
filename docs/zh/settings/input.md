# Input

Input（输入框）设置用于统一调整商店中表单字段的尺寸、边框、圆角和阴影。它通常会影响搜索框、联系表单、邮件订阅、顾客信息及其他文本输入区域。

## 打开 Input 设置

1. 登录 Shopify 后台。
2. 进入 **在线商店 > 主题**。
3. 找到当前使用的主题，点击 **自定义**。
4. 点击左侧的 **主题设置**。
5. 选择 **Input**。

![Input 设置](/images/zh/settings/input.svg)

## Spacing

Spacing（间距）用于控制输入内容与输入框边缘之间的空间，也就是输入框的内边距。

### Input horizontal padding

**Input horizontal padding** 设置输入框左右两侧的内边距。

- 数值越大，文字与左右边缘之间的留白越多，输入框看起来更宽松。
- 数值越小，输入框内部更紧凑，可以为较窄的布局保留更多空间。

水平内边距会影响占位文字、用户输入内容以及输入框内部图标的位置。调整后应检查搜索框和较长的表单提示是否完整显示。

### Input vertical padding

**Input vertical padding** 设置输入框顶部和底部的内边距。

- 数值越大，输入框越高，触控区域更大。
- 数值越小，输入框高度越低，表单布局更加紧凑。

::: tip 移动端建议
输入框需要保留足够的高度，方便用户在触摸屏上点击。不要为了压缩页面空间而将垂直内边距设置得过小。
:::

## Border

Border（边框）用于控制输入框轮廓的粗细和圆角。

### Input border thickness

**Input border thickness** 设置输入框边框的宽度。

- 较细的边框风格简洁，适合大多数表单。
- 较粗的边框可以增强输入区域与背景之间的区分。
- 设置为 `0px` 时不显示边框，需要通过背景颜色或阴影帮助用户识别输入区域。

输入框边框颜色由当前配色方案中的 **Input border** 控制。调整粗细后，请检查默认状态和键盘焦点状态是否容易区分。

### Input radius

**Input radius** 设置输入框四个角的圆角大小。

- 较小的数值让输入框更方正，风格更加利落。
- 中等数值适合简洁、通用的界面风格。
- 较大的数值会让输入框更加圆润、柔和。

::: info 提示
建议让输入框圆角与按钮圆角保持协调，使搜索、订阅和表单区域看起来更加统一。
:::

## Shadow

Shadow（阴影）用于增强输入框与页面背景之间的层次，也可以用于制作轻微悬浮或硬边框效果。

### Input shadow opacity

**Input shadow opacity** 设置输入框阴影的不透明度。

- `0%` 表示完全不显示阴影。
- 数值越大，阴影越明显。

建议先调整透明度，再设置偏移和模糊度。透明度为 `0%` 时，其他阴影参数不会产生可见效果。

### Input shadow horizontal offset

**Input shadow horizontal offset** 设置阴影在水平方向上的偏移距离。

- 正数让阴影向右移动。
- 负数让阴影向左移动。
- `0px` 表示阴影在水平方向与输入框对齐。

### Input shadow vertical offset

**Input shadow vertical offset** 设置阴影在垂直方向上的偏移距离。

- 正数让阴影向下移动。
- 负数让阴影向上移动。
- `0px` 表示阴影在垂直方向与输入框对齐。

### Input shadow blur

**Input shadow blur** 设置阴影边缘的模糊程度。

- 较小的数值产生边缘清晰、较硬的阴影。
- 较大的数值产生柔和、扩散的阴影。
- `0px` 会显示没有模糊效果的硬阴影。

::: warning 注意
阴影不应代替清晰的输入框边界。阴影过深或偏移过大可能让表单显得杂乱，也可能使输入框看起来像按钮。
:::

## 输入框颜色与焦点

输入框的文字、背景、边框和占位文字颜色需要在 **主题设置 > Colors** 的对应配色方案中设置：

- **Input text**：用户输入内容的文字颜色。
- **Input background**：输入框背景颜色。
- **Input border**：输入框边框颜色。
- **Input placeholder**：占位提示文字颜色。

键盘选中输入框时显示的焦点环宽度和偏移，则在 **主题设置 > Focus** 中设置。

::: info 可访问性提示
占位文字不能代替字段标签。表单应保留清晰的字段名称，并确保输入文字、占位文字、边框和背景之间具有足够的颜色对比。
:::

## 推荐风格

可以根据商店的整体设计使用以下设置思路：

- **简洁表单**：使用适中内边距、细边框和轻微圆角，并关闭阴影。
- **柔和圆润**：使用较大圆角和充足的垂直内边距，搭配透明度较低的柔和阴影。
- **清晰描边**：使用明显边框、较小圆角，并关闭阴影。
- **轻微悬浮**：使用细边框，为阴影设置较低透明度、轻微向下偏移和适度模糊。

## 保存与检查

完成设置后，点击主题编辑器右上角的 **保存**。建议检查以下内容：

- 搜索框的占位文字和输入内容是否完整显示。
- 联系表单中的短字段和多行文本区域是否风格一致。
- 邮件订阅输入框与提交按钮的高度是否协调。
- 输入框在浅色和深色配色方案中是否容易识别。
- 使用 `Tab` 键选中输入框时，焦点环是否清晰可见。
- 错误提示出现时，输入框和错误文字是否容易理解。
- 移动端点击输入框时是否方便，文字是否被图标遮挡。
- 长文本、自动填充内容和不同语言文字是否正常显示。

::: tip 建议
完成外观检查后，请实际填写并提交搜索、联系和订阅表单，确认输入、清除、自动填充和错误提示等状态都能正常工作。
:::
