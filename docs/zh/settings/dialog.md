# Dialog

Dialog（对话框）设置用于统一调整弹窗类界面的边框、圆角和阴影。它通常会影响快速查看、搜索弹层、邮件订阅弹窗及其他模态窗口；具体应用位置取决于当前页面所使用的功能。

## 打开 Dialog 设置

1. 登录 Shopify 后台。
2. 进入 **在线商店 > 主题**。
3. 找到当前使用的主题，点击 **自定义**。
4. 点击左侧的 **主题设置**。
5. 选择 **Dialog**。

![Dialog 设置](/images/zh/settings/dialog.svg)

## Border

Border（边框）用于控制对话框轮廓的粗细和圆角。

### Dialog border thickness

**Dialog border thickness** 设置对话框边框的宽度。

- 较小的数值会形成轻薄、简洁的轮廓。
- 较大的数值会让对话框边界更加明显。
- 设置为 `0px` 时不显示边框，主要依靠背景和阴影区分对话框与页面内容。

如果对话框背景与页面背景颜色接近，可以使用边框或阴影增强层次。边框颜色通常由当前配色方案中的边框颜色控制。

### Dialog radius

**Dialog radius** 设置对话框四个角的圆角大小。

- 较小的数值让对话框更方正，风格更加简洁、正式。
- 中等数值能够在利落感与柔和感之间取得平衡。
- 较大的数值会让对话框更圆润，适合轻松、亲切的品牌风格。

::: tip 建议
Dialog radius 建议与按钮、输入框和卡片的圆角风格保持协调，但不需要完全使用相同数值。
:::

## Shadow

Shadow（阴影）用于让对话框在页面内容上方形成悬浮效果，并帮助用户识别当前需要操作的区域。

### Dialog shadow opacity

**Dialog shadow opacity** 设置阴影的不透明度。

- `0%` 表示完全不显示阴影。
- 数值越大，阴影颜色越明显，对话框与背景之间的层次越强。

建议先设置透明度，再调整偏移和模糊度。透明度为 `0%` 时，其他阴影参数不会产生可见效果。

### Dialog shadow horizontal offset

**Dialog shadow horizontal offset** 设置阴影在水平方向上的偏移距离。

- 正数让阴影向右移动。
- 负数让阴影向左移动。
- `0px` 表示阴影在水平方向与对话框对齐。

为了让弹窗呈现自然、居中的悬浮效果，通常可以将水平偏移保持在较小数值或 `0px`。

### Dialog shadow vertical offset

**Dialog shadow vertical offset** 设置阴影在垂直方向上的偏移距离。

- 正数让阴影向下移动，能够模拟对话框浮在页面上方的效果。
- 负数让阴影向上移动。
- `0px` 表示阴影在垂直方向与对话框对齐。

使用轻微的向下偏移通常能够产生比较自然的层次感。

### Dialog shadow blur

**Dialog shadow blur** 设置阴影边缘的模糊程度。

- 较小的数值会产生边缘清晰、较硬的阴影。
- 较大的数值会产生柔和、扩散范围更大的阴影。
- `0px` 会显示没有模糊效果的硬阴影。

::: warning 注意
阴影透明度、偏移和模糊度会共同决定最终效果。阴影过深或范围过大可能使页面显得沉重，也可能遮挡对话框周围的内容。
:::

## 推荐风格

可以根据商店的整体设计使用以下设置思路：

- **简洁无阴影**：使用细边框和适中圆角，将阴影透明度设为 `0%`。
- **轻柔悬浮**：使用适中圆角、较低阴影透明度、轻微向下偏移和较大的模糊度。
- **清晰轮廓**：使用明显边框和较小圆角，搭配很轻或不使用阴影。
- **现代圆润**：使用较大圆角、无边框或细边框，并添加柔和阴影。

::: info 配色提示
边框和阴影只能帮助建立层次。还需要确保对话框背景、文字、按钮与页面遮罩之间具有足够的颜色对比。
:::

## 保存与检查

完成设置后，点击主题编辑器右上角的 **保存**。建议打开商店中实际使用的不同对话框并检查：

- 对话框与页面背景之间的边界是否清晰。
- 圆角是否与按钮、输入框和卡片风格协调。
- 阴影是否自然，是否遮挡附近的重要内容。
- 浅色和深色配色方案下是否都能清楚显示。
- 桌面端与移动端的对话框是否完整显示在屏幕内。
- 标题、正文、输入框、按钮和关闭图标是否容易识别与操作。
- 使用键盘时，焦点是否能够正常进入和离开对话框。

::: tip 建议
完成外观检查后，还应测试打开、关闭、提交和取消等操作，确保样式调整没有影响对话框的正常使用。
:::
