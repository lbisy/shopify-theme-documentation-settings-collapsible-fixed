# Button

Button（按钮）设置用于统一调整商店中按钮的尺寸、边框、圆角和阴影。这里的设置通常会影响“添加到购物车”“立即购买”“提交”和“了解更多”等按钮。

## 打开 Button 设置

1. 登录 Shopify 后台。
2. 进入 **在线商店 > 主题**。
3. 找到当前使用的主题，点击 **自定义**。
4. 点击左侧的 **主题设置**。
5. 选择 **Button**。

![Button 设置](/images/zh/settings/button.svg)

## Spacing

Spacing（间距）用于调整按钮文字与按钮边缘之间的空间，也就是按钮的内边距。

### Button horizontal padding

**Button horizontal padding** 设置按钮左右两侧的内边距。

- 数值越大，按钮越宽，文字两侧的留白越多。
- 数值越小，按钮越紧凑，适合空间有限的区域。

水平内边距会与按钮文字长度共同决定按钮的最终宽度。调整后应重点检查文字较长的按钮，以及移动端同一行显示多个按钮的场景。

### Button vertical padding

**Button vertical padding** 设置按钮顶部和底部的内边距。

- 数值越大，按钮越高，视觉效果更突出，也更容易点击。
- 数值越小，按钮高度越低，整体布局更加紧凑。

::: tip 建议
按钮需要保留足够的可点击区域。不要为了节省空间而将垂直内边距设置得过小，尤其是在移动端。
:::

## Border

Border（边框）用于控制按钮轮廓的粗细和圆角。

### Button border thickness

**Button border thickness** 设置按钮边框的宽度。

- 较细的边框风格简洁，适合大多数按钮。
- 较粗的边框更加醒目，适合强调描边按钮或具有强烈视觉风格的商店。
- 设置为 `0px` 时，按钮不显示边框。

边框颜色由当前配色方案中的按钮边框颜色控制。调整粗细后，建议同时检查主按钮和次要按钮的显示效果。

### Button radius

**Button radius** 设置按钮四个角的圆角大小。

- 较小的数值让按钮更方正，风格更加利落、正式。
- 中等数值可以在简洁感与柔和感之间取得平衡。
- 较大的数值会让按钮更圆润，适合轻松、亲切的品牌风格。

::: info 提示
Button radius 只调整按钮的外观，不会改变按钮文字或点击功能。
:::

## Shadow

Shadow（阴影）用于为按钮增加立体感，并让按钮与页面背景形成更明显的层次。

### Button shadow opacity

**Button shadow opacity** 设置阴影的不透明度。

- `0%` 表示完全不显示阴影。
- 数值越大，阴影越明显。

建议先设置阴影透明度，再调整偏移和模糊度。透明度为 `0%` 时，即使其他阴影参数不为零，阴影也不可见。

### Button shadow horizontal offset

**Button shadow horizontal offset** 设置阴影在水平方向上的偏移距离。

- 正数让阴影向右移动。
- 负数让阴影向左移动。
- `0px` 表示阴影在水平方向与按钮对齐。

### Button shadow vertical offset

**Button shadow vertical offset** 设置阴影在垂直方向上的偏移距离。

- 正数让阴影向下移动。
- 负数让阴影向上移动。
- `0px` 表示阴影在垂直方向与按钮对齐。

### Button shadow blur

**Button shadow blur** 设置阴影边缘的模糊程度。

- 较小的数值会产生清晰、硬朗的阴影。
- 较大的数值会产生柔和、扩散的阴影。
- `0px` 会显示边缘清晰的阴影。

::: warning 注意
过深、偏移过大或模糊过强的阴影可能会影响文字阅读，并让页面显得杂乱。建议在不同背景颜色和不同按钮状态下检查效果。
:::

## 推荐风格

可以根据商店的品牌风格使用以下设置思路：

- **简洁扁平**：使用适中的内边距、较细边框和圆角，并将阴影透明度设为 `0%`。
- **柔和圆润**：增加圆角与垂直内边距，使用透明度较低、模糊度较高的轻微阴影。
- **硬朗描边**：使用较小圆角、较明显的边框，并关闭阴影。
- **立体强调**：使用适中的圆角，为阴影设置轻微的水平和垂直偏移，并控制较低的透明度。

## 保存与检查

完成设置后，点击主题编辑器右上角的 **保存**。建议检查以下内容：

- 主按钮和次要按钮的尺寸是否协调。
- 较长的按钮文字是否完整显示。
- 移动端按钮是否容易点击，是否超出屏幕宽度。
- 描边按钮的边框是否清晰。
- 圆角是否符合品牌的整体视觉风格。
- 阴影在浅色和深色背景上是否自然。
- 商品页、购物车和表单中的按钮样式是否保持一致。

::: tip 建议
按钮样式应优先保证可识别性和可点击性。完成设置后，建议实际测试添加到购物车、立即购买和表单提交等关键操作。
:::
