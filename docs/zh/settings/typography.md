# Typography

Typography（文字排版）设置用于统一管理商店的字体、字号、行高、字间距和大小写样式。合理的文字设置可以强化品牌风格，同时提升桌面端与移动端的阅读体验。

## 打开 Typography 设置

1. 登录 Shopify 后台。
2. 进入 **在线商店 > 主题**。
3. 找到当前使用的主题，点击 **自定义**。
4. 点击左侧的 **主题设置**。
5. 选择 **Typography**。

![Typography 设置](/images/zh/settings/typography.jpg)

## Font family

**Font family** 用于分别设置标题和正文所使用的字体。你可以直接选择 Shopify 字体库中的字体，也可以通过字体文件 URL 使用自定义字体。

### Heading

**Heading** 控制网站标题使用的字体，例如页面标题、区块标题和商品标题。

- **Use custom font URL for headings**：开启后，标题将使用下方填写的自定义字体。
- **Custom heading font family name**：填写自定义标题字体的名称，例如 `Neue Montreal`。该名称用于识别字体，建议与字体本身的名称保持一致。
- **Custom heading font file URL (.woff2 recommended)**：填写字体文件的完整 URL。建议使用 `.woff2` 格式，以兼顾文件大小和浏览器兼容性。
- **Font**：未启用自定义字体时，从字体列表中选择标题字体。

### Body

**Body** 控制网站正文使用的字体，例如商品描述、段落、菜单和按钮文字。

- **Use custom font URL for body**：开启后，正文将使用下方填写的自定义字体。
- **Custom body font family name**：填写自定义正文字体的名称。
- **Custom body font file URL (.woff2 recommended)**：填写正文所用字体文件的完整 URL。
- **Font**：未启用自定义字体时，从字体列表中选择正文字体。

::: tip 自定义字体建议
将字体文件上传到 Shopify 的 **内容 > 文件** 后，复制文件 URL 并粘贴到对应输入框。请确认你拥有该字体的网页使用授权，并尽量使用 `.woff2` 文件以减少页面加载时间。
:::

::: warning 注意
只有在打开对应的 **Use custom font URL** 开关后，自定义字体名称和文件 URL 才会生效。如果字体没有正常显示，请检查 URL 是否完整、文件是否可以公开访问，以及字体名称是否填写正确。
:::

## Heading

此部分用于调整全站标题文字的视觉样式。

- **Weight**：设置标题的粗细。较粗的标题更醒目，较细的标题则更加轻盈。
- **Line height**：设置标题的行高。多行标题需要保留足够的行间距，避免文字显得拥挤。
- **Letter spacing**：设置标题字符之间的间距。稍紧的字间距可以让大标题更紧凑，较宽的字间距则会增强呼吸感。
- **Text transform**：控制标题的英文大小写转换，例如保持原样、全部大写或全部小写。

::: tip 建议
标题的主要作用是建立清晰的内容层级。调整字重和字间距后，请重点检查较长的商品标题以及移动端的多行标题。
:::

## Subtitle

**Subtitle** 用于设置副标题或辅助标题的样式。

- **Font family**：选择副标题使用标题字体还是正文字体。
- **Weight**：设置副标题的粗细。
- **Line height**：设置副标题的行高。
- **Letter spacing**：设置副标题的字符间距。
- **Text transform**：控制副标题的英文大小写转换。

副标题通常用于补充主标题，因此建议在字号或字重上与主标题形成区别，同时保持足够的可读性。

## Body

**Body** 用于控制全站正文的基础排版。

- **Font size (desktop)**：设置桌面端的正文基础字号。
- **Font size (mobile)**：设置移动端的正文基础字号。
- **Line height**：设置正文的行高。
- **Letter spacing**：设置正文字符之间的间距。
- **Text transform**：控制正文的英文大小写转换。

桌面端和移动端字号可以分别设置。移动端屏幕较窄，应避免字号过小；正文行高也应留出足够空间，方便用户连续阅读商品说明、博客文章和帮助内容。

::: warning 注意
不建议对大段正文使用全部大写。全部大写更适合短标签或按钮文字，应用于长段落时可能降低阅读效率。
:::

## Scale

Scale（缩放）用于在现有排版比例的基础上，统一放大或缩小标题与正文。

- **Heading scale**：调整全站标题文字的整体缩放比例。
- **Body scale**：调整全站正文及相关文字的整体缩放比例。

`100%` 表示使用主题原始大小。高于 `100%` 会整体放大，低于 `100%` 会整体缩小。

Scale 适合在字体和基础字号已经确定后进行整体微调。建议先完成字体、字号和行高设置，再调整缩放比例，避免多项设置叠加后导致文字过大或过小。

## 检查与保存

完成设置后，点击主题编辑器右上角的 **保存**，并检查以下内容：

- 页面标题、区块标题与正文之间的层级是否清晰。
- 桌面端和移动端的正文是否容易阅读。
- 长标题是否出现过度换行或文字拥挤。
- 菜单、按钮和商品卡片中的文字是否完整显示。
- 自定义字体加载时是否出现明显跳动或加载缓慢。
- 中文与英文内容在当前字体下是否都能正常显示。

::: info 提示
部分英文字体不包含中文字形。商店包含中文内容时，请确认所选字体支持中文，或检查浏览器回退字体的显示效果。
:::
