# 商品页面

Product 分区是商品详情页的主要内容区域，用于展示商品图片、视频、标题、价格、变体、购买按钮及其他商品信息。通过本分区可以调整媒体画廊布局、桌面端媒体尺寸、缩略图位置、图片放大效果和上下间距。

## 打开 Product 分区

1. 登录 Shopify 后台。
2. 进入 **在线商店 → 主题**。
3. 在当前使用的主题上点击 **自定义**。
4. 使用编辑器顶部的页面选择器打开 **商品 → 默认商品**。
5. 在左侧模板列表中点击 **Product** 分区。
6. 完成设置后点击右上角的 **保存**。

![商品页面设置](/images/zh/product-page.svg)

不同商品需要使用不同布局时，可以创建新的商品模板，并在 Shopify 后台将模板分配给对应商品。

## Color scheme

**Color scheme** 用于设置 Product 分区的背景、文字、按钮、图标和其他界面元素的配色。点击 **编辑方案** 可以进入全局颜色设置。

::: warning 注意
配色方案属于全局设置。修改方案本身会同时影响其他使用该方案的分区，请检查商品标题、价格、按钮及图片区域与背景之间的对比度。
:::

## Image layout

**Image layout** 设置商品媒体在详情页中的排列方式。选择 **Thumbnails** 后，当前媒体以大图显示，其他媒体以缩略图形式排列，顾客可以点击缩略图切换主图。

商品媒体可能包含图片、视频、外部视频和 3D 模型。调整布局后，应逐一检查各种媒体类型能否正常加载、切换和播放。

## Thumbnail position

**Thumbnail position** 设置缩略图相对于主媒体的位置，仅在 **Image layout** 设为 **Thumbnails** 时生效。

- **Left**：缩略图纵向排列在主媒体左侧，适合横向空间充足的桌面布局。
- **Bottom**：缩略图横向排列在主媒体下方，可以为主图保留更多宽度。

缩略图数量较多时，应检查列表是否可以继续浏览，以及当前选中状态是否清晰可见。

## Desktop media width

**Desktop media width** 设置桌面端媒体区域在 Product 分区中占用的宽度比例。增大数值会突出商品图片，同时压缩右侧商品信息区域；减小数值则会为标题、变体和购买操作留出更多空间。

调整时应重点检查以下内容：

- 较长的商品标题是否出现过多换行。
- 变体选择器和购买按钮是否保持完整。
- 图片是否足够清晰，且不会挤压商品信息。
- 宽屏和普通笔记本屏幕中的左右比例是否协调。

该设置主要影响桌面端，移动端通常会按照主题的响应式规则重新排列媒体和商品信息。

## Media aspect ratio

**Media aspect ratio** 设置商品媒体容器的宽高比例。选择固定比例后，图片会在统一的媒体区域中展示，超出容器的部分可能被裁切。

**Portrait (3:4)** 适合竖版商品图片，能够为服装、家具及其他纵向构图的商品保留较多展示高度。选择比例时，应结合商品原图构图，并确认商品主体、文字和重要细节没有被裁掉。

::: tip 图片建议
同一商品的媒体最好使用一致或相近的宽高比例。这样可以减少切换媒体时的页面跳动，并让缩略图列表更加整齐。
:::

## Desktop media height

**Desktop media height** 控制桌面端媒体画廊的最大高度。

- **Balanced**：限制画廊高度，使媒体区域与右侧商品信息保持相对均衡，适合商品标题或购买选项较多的页面。
- **Original**：媒体高度跟随媒体宽度和所选宽高比，不额外限制画廊高度。

媒体数量较多或图片较高时，使用受限高度可以减少顾客查看购买信息所需的滚动距离。修改后请检查主图是否完整、缩略图是否容易操作，以及页面首屏能否看到关键购买信息。

## Bottom visible thumbnails

**Bottom visible thumbnails** 设置媒体画廊底部一行直接显示的缩略图数量。该设置仅在以下条件同时满足时生效：

- **Image layout** 设为 **Thumbnails**。
- **Thumbnail position** 设为 **Bottom**。

显示数量越多，单个缩略图越小；显示数量越少，缩略图更容易辨认，但顾客可能需要更多操作才能浏览全部媒体。应根据商品媒体数量和页面宽度选择合适的数量。

## Magnifier preview size

**Magnifier preview size** 设置图片放大镜预览区域的尺寸，单位为 `px`。数值越大，顾客一次可以看到的局部放大范围越大，同时也会遮挡更多原图区域。

此功能适合展示材质纹理、工艺细节、接口和其他需要近距离查看的内容。请使用分辨率足够高的商品图片，否则放大后可能出现模糊或锯齿。

## Zoom overlay

Zoom overlay 用于设置打开媒体放大视图时的页面遮罩。

### Zoom overlay color

**Zoom overlay color** 设置放大视图周围遮罩的颜色。深色遮罩可以增强媒体的视觉焦点，浅色遮罩则更适合整体明亮的页面风格。

### Zoom overlay opacity

**Zoom overlay opacity** 设置遮罩的不透明度：

- 较低的数值会保留更多页面背景内容。
- 较高的数值会弱化背景，让顾客更专注于放大的商品媒体。
- 设置过高时，应确认关闭按钮、切换按钮和其他操作控件仍然清晰可见。

遮罩颜色与透明度需要配合调整。完成设置后，应检查浅色图片、深色图片、视频和 3D 模型在放大视图中的显示效果。

## Padding

- **Padding Top**：设置 Product 分区顶部的留白，单位为 `px`。
- **Padding Bottom**：设置 Product 分区底部的留白，单位为 `px`。

顶部间距会影响商品内容与页眉之间的距离，底部间距会影响 Product 分区与相关商品、对比表等后续分区之间的距离。调整时应同时检查桌面端和移动端的页面衔接。

## 商品信息区块

Product 分区中的商品信息由多个区块组成。可以在左侧模板列表中点击区块进行配置，也可以拖动区块改变显示顺序。

- **Vendor**：显示商品供应商或品牌名称。
- **Title**：显示商品标题。
- **Inventory notice**：显示库存状态或低库存提示。
- **Description**：显示商品描述。
- **Callout**：突出展示配送、公益承诺或其他简短信息。
- **Price**：显示价格、原价及促销价格。
- **Quantity selector**：允许顾客调整购买数量。
- **Variant picker**：允许顾客选择颜色、尺寸等商品变体。
- **Buy buttons**：显示加入购物车和立即购买按钮。
- **Icon with text**：使用图标和短文案展示配送、退换货或安全结账等信息。
- **Share**：提供商品分享入口。
- **Spacer**：在相邻区块之间增加留白。
- **Collapsible tab**：以可展开面板展示尺寸、材质、护理方式或配送政策等补充信息。

并非所有商品都需要显示全部区块。应优先保留影响购买决策的信息，并确保价格、变体和购买按钮容易找到。

## 相关分区

商品模板还可以包含以下独立分区：

- **Comparison Table**：对比商品特征或不同型号。
- **Related products**：向顾客推荐相关商品。

这些分区不属于 Product 主分区的内容区块，可以独立配置、排序、隐藏或移除。

## 保存与检查

完成设置后点击主题编辑器右上角的 **保存**，并检查以下内容：

- 切换所有图片、视频和 3D 模型，确认媒体加载及操作正常。
- 检查缩略图位于左侧和底部时的排列、选中状态和浏览操作。
- 确认主图在所选宽高比下没有错误裁切商品主体。
- 测试放大镜和媒体放大视图，确认清晰度、遮罩和关闭操作正常。
- 检查长标题、促销价格、多个变体和低库存提示的排版。
- 测试数量选择、变体切换、加入购物车和立即购买按钮。
- 确认缺货或不可用变体不能被错误加入购物车。
- 分别预览桌面端和移动端，检查媒体顺序、按钮宽度、触控区域和上下间距。
- 检查 Comparison Table、Related products 等后续分区与 Product 分区之间的距离。
