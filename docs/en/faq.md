# FAQ and Troubleshooting

## The homepage is empty
Add sections to the homepage template and assign images, products, or collections.

## The menu does not appear
Create the menu in Shopify Admin, then assign it in Header or Footer settings.

## Product information is incorrect
Check product media, variants, price, inventory, and metafields in Shopify Admin.

## Changes are not visible
Confirm you edited the correct theme and template, click Save, and test in a private browser window.

## The uploaded theme reports an error
Confirm the ZIP contains the correct theme root and required Shopify theme files.
