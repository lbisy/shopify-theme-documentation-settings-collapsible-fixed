# Theme Installation

1. Sign in to Shopify Admin.
2. Go to **Online Store → Themes**.
3. Click **Add theme → Upload zip file**.
4. Upload the original theme ZIP package.
5. Wait for Shopify to finish processing.
6. Click **Customize** to configure the unpublished theme.
7. Test the storefront before clicking **Publish**.

![Theme upload page](/images/en/theme-upload.jpg)

## Important

Do not unzip and re-compress the theme unless necessary. An incorrect ZIP structure may cause Shopify to report missing theme files.
