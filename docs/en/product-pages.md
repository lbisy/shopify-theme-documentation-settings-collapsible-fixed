# Product Pages

The **Product** section is the main product-page area. It displays media, title, price, variants, purchase actions, and supporting information. Use it to configure the gallery, desktop media size, thumbnails, zoom, and spacing.

## Open the Product section

1. Go to **Shopify admin > Online Store > Themes**.
2. Click **Customize** on the current theme.
3. In the page selector, open **Products > Default product**.
4. Select the **Product** section and configure it.
5. Click **Save**.

![Product page settings](/images/en/product-page.svg)

Create additional product templates when different products require different layouts, then assign them in Shopify admin.

## Color scheme

Controls the section background, text, buttons, icons, and other interface elements. **Edit scheme** opens global colors; changing the scheme also affects other sections that use it. Verify contrast for titles, prices, controls, and media.

## Media gallery

### Image layout

With **Thumbnails**, the active item appears large and other media appear as selectable thumbnails. Product media can include images, videos, external video, and 3D models; test every type.

### Thumbnail position

Available when the layout is **Thumbnails**:

- **Left**: vertical thumbnails beside the main media.
- **Bottom**: horizontal thumbnails below, leaving more width for the main image.

If there are many items, confirm the list can be browsed and the selected state is clear.

### Desktop media width

Increasing the value emphasizes imagery but reduces space for product information. Decreasing it provides more room for long titles, variants, and buttons. Check wide desktops and typical laptops; mobile uses its own responsive layout.

### Media aspect ratio

A fixed ratio creates a consistent container but may crop image edges. **Portrait (3:4)** suits vertically composed products. Keep media ratios similar to reduce layout movement and avoid cropping products, labels, or important detail.

### Desktop media height

- **Balanced** limits gallery height so media and purchase information remain visually balanced.
- **Original** follows the selected width and aspect ratio without an extra height limit.

For tall or numerous images, Balanced can keep key purchase controls closer to the first viewport.

### Bottom visible thumbnails

Controls how many thumbnails appear in the bottom row when the layout is **Thumbnails** and position is **Bottom**. More items create smaller thumbnails; fewer items are easier to recognize but require more browsing.

## Magnifier and zoom

**Magnifier preview size** sets the magnified area in `px`. Larger previews show more detail but cover more of the original image. Use sufficiently high-resolution files.

**Zoom overlay color** and **Zoom overlay opacity** control the backdrop around expanded media. Higher opacity focuses attention on the media; ensure close and navigation controls stay visible. Test light and dark images, video, and 3D models.

## Padding

**Padding Top** controls space below the header. **Padding Bottom** controls separation from following sections such as recommendations or comparison tables. Check both desktop and mobile.

## Product information blocks

Blocks can be configured and reordered in the left panel:

- **Vendor**
- **Title**
- **Inventory notice**
- **Description**
- **Callout**
- **Price**
- **Quantity selector**
- **Variant picker**
- **Buy buttons**
- **Icon with text**
- **Share**
- **Spacer**
- **Collapsible tab**

Keep information that affects purchase decisions and make price, variants, and purchase actions easy to find. Not every product needs every block.

## Related sections

**Comparison Table** and **Related products** are independent sections in the product template. They can be configured, reordered, hidden, or removed separately.

## Save and test

- Switch through every image, video, and 3D model.
- Test left and bottom thumbnails, selection state, and browsing.
- Check cropping at the selected ratio.
- Test magnifier, expanded zoom, backdrop, and closing.
- Review long titles, sale pricing, multiple variants, and low-stock notices.
- Test quantity, variants, add to cart, and buy now.
- Ensure sold-out or unavailable variants cannot be purchased.
- Review media order, button width, touch targets, and spacing on desktop and mobile.
- Check spacing before comparison and recommendation sections.
