# Header, Announcement Bar, and Footer

The Header, Announcement Bar, and Footer are shared across the storefront. Changes usually affect every page using the same header or footer group, so preview desktop and mobile before saving.

## Start configuring

1. Go to **Shopify admin > Online Store > Themes**.
2. Click **Customize** on the current theme.
3. Find the **Header** or **Footer** group.
4. Select **Announcement Bar**, **Header**, or **Footer**.
5. Configure it and click **Save**.

Create navigation under **Content > Menus** (or **Online Store > Navigation**) before assigning it. Upload the logo under Theme settings.

## Header

The header contains the logo, primary navigation, search, customer account, and cart entry points.

![Header settings](/images/en/header-settings.svg)

### Header and color

- **Logo Position**: places the logo within the header.
- **Edit your logo in theme settings**: upload or replace the logo. If none is uploaded, the store name usually appears.
- **Color Scheme**: header background, text, and icons.
- **Menu Color Scheme**: desktop dropdown or mega-menu colors.
- **Mobile Menu Color Scheme**: mobile drawer colors.
- **Edit scheme**: opens global color settings and can affect other areas using the same scheme.

Check contrast in the default header, expanded desktop menu, and mobile menu.

### Menu

Use **Menu** to select a navigation menu such as **Main menu**. Edit its hierarchy and links in Shopify menu management.

### Menu Type

- **Menu Type**: choose a presentation such as **Super Menu**.
- **Desktop menu trigger**:
  - **Hover** opens submenus when the pointer passes over them.
  - **Click** requires a deliberate click and reduces accidental opening.
- **Super Menu Watermark**: short decorative brand text.
- **Enable Product Series Tabs**: displays collection tabs in the super menu.
- **Series Collection 1 / 2 / 3**: selects collections for those tabs.

Keep menu levels clear and collection names short. Confirm that every link and collection is published to the current sales channel.

## Announcement Bar

Use the bar above the header for shipping notices, promotions, opening information, or important policy links.

1. Expand **Announcement Bar**.
2. Add or select an announcement block.
3. Enter concise text and an optional link.
4. Check contrast and mobile wrapping.

Focus on one primary message. If several announcements rotate, put the most important first and remove expired promotions promptly.

## Footer

The footer provides brand information, supporting navigation, and follow actions.

![Footer settings](/images/en/footer-settings.svg)

### Appearance and brand

- **Color Scheme**: footer colors.
- **Background Image**: a wide image that keeps overlaid text readable.
- **Brand Name**: rich text such as `Eccenova®`.
- **Brush Texture Image**: a transparent PNG or WebP decorative texture.
- **Location Text**: brand or store location, such as `Shibuya, JP`.

### Right Content

- **Heading**
- **Description**
- **Link** to a collection, product, page, or URL.
- **Link Text**

Leave both link fields blank when no call to action is needed.

### Follow on Shop

**Enable Follow on Shop button** controls the Shop follow button. Confirm the store meets Shop-channel requirements and test the button in preview.

## Before publishing

- Confirm brand colors, logo, and readable contrast.
- Test every primary, dropdown, super-menu, and footer link.
- Test the selected Hover or Click trigger.
- Verify series collections are published and correct.
- Check responsive cropping of background and texture images.
- Review menu opening, text wrapping, icon positions, and footer layout on desktop and mobile.
- Remove expired announcements and click **Save**.
