# Header, Announcement Bar, and Footer

Create menus in Shopify Admin before assigning them to the Header or Footer.

## Header
Configure the logo, navigation, icons, sticky behavior, and mobile menu options.

## Announcement Bar
Use it for shipping notices, promotions, or store updates.

## Footer
Add menus, brand information, social links, newsletter content, and policy links.
