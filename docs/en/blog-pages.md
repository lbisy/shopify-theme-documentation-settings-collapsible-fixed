# Blog Pages

The blog page displays article lists. Create blogs and articles in Shopify admin first, then configure the page heading, description, category tabs, rotating badge, articles per page, and spacing.

![Blog page settings](/images/en/blog-article.svg)

## Create blogs and articles

1. Go to **Shopify admin > Content > Blog posts**.
2. Create a post and enter its title and body.
3. Add a featured image, excerpt, author, tags, and search-engine information as needed.
4. Assign the post to a blog, creating one first if necessary.
5. Make the post visible and save it.
6. Add the blog to a header or footer menu under **Content > Menus**.

Use a consistent featured-image ratio and concise excerpts so article cards align well.

## Open the blog template

1. Open **Online Store > Themes > Customize**.
2. In the page selector, choose **Blogs > Default blog**.
3. Select the **Blog** section.
4. Configure it and click **Save**.

Optional sections such as **Blog Stories** and **Newsletter Banner** can be reordered, hidden, or removed independently.

## Header

- **Header Color Scheme**: colors for the title area.
- **Heading size**: for example, **3XL**.
- **Header**: the main title, such as `Blog`.
- **Description**: a short rich-text introduction.

Editing a global color scheme also changes other sections that use it. Check contrast among the site header, blog header, and article cards.

## Subtitle and tabs

- **Subtitle size**: size of category or tag controls.
- **Tab Control Color Scheme**: tab colors and selected state.

`All (6)` means all categories and the current article count. Confirm selected text and underlines remain visible.

## Rotating badge

- **Rotate Badge Text**: outer circular text, such as `PRODUCT GUARANTEE NATURAL`.
- **Rotate Badge Percent**: short center text, such as `100%`.

Keep outer text concise and check that it wraps cleanly without overlapping the heading.

## Article list

- **Articles per page**: the number shown before pagination.
- **Link Text**: the article-card link, such as `Read More`.

Test desktop, mobile, and pagination whenever the total article count exceeds the page limit.

## Padding

Set **Padding Top** and **Padding Bottom** in `px`. Start with `32 px`; increase top spacing if the header feels crowded or reduce bottom spacing if the next section is too far away.

## Example configuration

| Setting | Example |
| --- | --- |
| Heading size | `3XL` |
| Header | `Blog` |
| Subtitle size | `XL` |
| Rotate Badge Text | `PRODUCT GUARANTEE NATURAL` |
| Rotate Badge Percent | `100%` |
| Articles per page | `3` |
| Link Text | `Read More` |
| Padding | `32 px / 32 px` |

Replace sample copy with language and content appropriate to the store.

## Before publishing

- Verify blog assignments, visibility, titles, excerpts, images, authors, and dates.
- Check heading, description, tabs, and badge for overlap or poor wrapping.
- Keep card image ratios, heights, and reading links consistent.
- Test tabs, cards, links, and pagination.
- Preview image cropping, type size, spacing, and columns on desktop and mobile.
- Confirm the navigation link opens the correct blog and save all changes.
