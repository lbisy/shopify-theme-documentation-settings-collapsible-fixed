# Typography

The **Typography** settings manage fonts, size, line height, letter spacing, capitalization, and global scale. Good typography reinforces the brand while keeping desktop and mobile content easy to read.

## Open the Typography settings

Go to **Shopify admin > Online Store > Themes**, click **Customize**, then open **Theme settings > Typography**.

![Typography settings](/images/en/settings/typography.svg)

## Font family

Set fonts separately for headings and body text. Choose a Shopify font or supply a custom font-file URL.

### Heading

- **Use custom font URL for headings** enables the custom fields.
- **Custom heading font family name** identifies the font, for example `Neue Montreal`.
- **Custom heading font file URL (.woff2 recommended)** accepts the full public file URL.
- **Font** selects a library font when custom fonts are disabled.

### Body

The equivalent body settings control product descriptions, paragraphs, menus, and button labels.

::: tip Custom fonts
Upload the font under **Content > Files**, then paste its URL. Confirm that your license permits web use and prefer `.woff2` for size and browser compatibility.
:::

::: warning Note
The name and URL fields work only when the corresponding **Use custom font URL** switch is enabled. If a font fails, verify the complete URL, public access, and exact family name.
:::

## Heading

- **Weight** controls stroke thickness.
- **Line height** controls spacing between lines.
- **Letter spacing** controls the space between characters.
- **Text transform** preserves or changes English capitalization.

Check long product titles and multiline mobile headings after adjusting these settings.

## Subtitle

Choose whether subtitles use the heading or body font, then configure **Weight**, **Line height**, **Letter spacing**, and **Text transform**. Subtitles should remain distinct from main headings while staying readable.

## Body

- **Font size (desktop)**
- **Font size (mobile)**
- **Line height**
- **Letter spacing**
- **Text transform**

Mobile screens are narrow, so avoid very small type and provide comfortable line height for descriptions, articles, and help text.

::: warning Readability
Do not apply all-uppercase styling to long body copy. It is better suited to short labels and buttons.
:::

## Scale

- **Heading scale** changes all heading sizes proportionally.
- **Body scale** changes body and related text proportionally.

`100%` uses the theme's original size. Higher values enlarge text and lower values reduce it. Set font, base size, and line height first, then use scale for final overall adjustment.

## Save and check

Click **Save**, then review:

- The hierarchy between page titles, section headings, and body text.
- Readability on desktop and mobile.
- Wrapping of long headings.
- Menu, button, and product-card labels.
- Loading behavior of custom fonts.
- Both Chinese and English content.

::: info Language support
Some Latin fonts do not contain Chinese glyphs. If the store includes Chinese content, choose a compatible font or verify that the browser's fallback font displays correctly.
:::
