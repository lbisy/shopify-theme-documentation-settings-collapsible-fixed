# Social icon

Use **Social icon** settings to add links to your store's social profiles. The corresponding icons appear in supported theme areas, such as the footer or social-media sections.

## Open the settings

1. In Shopify admin, go to **Online Store > Themes**.
2. Find the current theme and click **Customize**.
3. Open **Theme settings > Social icon**.

![Social icon settings](/images/en/settings/social-icon.svg)

## Add social-profile links

Paste the complete URL for each personal or brand profile:

| Setting | Example |
| --- | --- |
| Facebook | `https://www.facebook.com/yourbrand` |
| Instagram | `https://www.instagram.com/yourbrand` |
| YouTube | `https://www.youtube.com/@yourbrand` |
| TikTok | `https://www.tiktok.com/@yourbrand` |
| X / Twitter | `https://x.com/yourbrand` |
| Snapchat | `https://www.snapchat.com/add/yourbrand` |
| Pinterest | `https://www.pinterest.com/yourbrand` |
| Tumblr | `https://yourbrand.tumblr.com` |
| Vimeo | `https://vimeo.com/yourbrand` |

::: warning Note
Enter a full address beginning with `https://`. Do not enter only a username or placeholder values such as `1`, `2`, or `3`, because the icon may open the wrong page.
:::

Leave platforms you do not use blank, then click **Save**.

## Check the result

On the storefront, confirm that:

- Every completed platform displays the correct icon.
- Each icon opens the correct profile.
- Blank platforms do not appear.
- Icons are arranged correctly on desktop and mobile.

::: tip Recommendation
Use official brand-profile links and check them periodically to ensure they remain valid.
:::
