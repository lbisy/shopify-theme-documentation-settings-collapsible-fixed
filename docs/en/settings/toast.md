# Toast

A toast is a temporary status message for success, warnings, errors, or information. The **Toast** settings control notification position, duration, shape, and shadow.

## Open the settings

1. In Shopify admin, go to **Online Store > Themes**.
2. Find the current theme and click **Customize**.
3. Open **Theme settings > Toast**.

![Toast settings](/images/en/settings/toast.svg)

## Basic settings

### Toast radius

Controls corner roundness. Higher values create softer corners; lower values produce a squarer appearance.

### Position

Controls where the toast appears. For example, **Top center** places it at the center of the top edge. Avoid covering header navigation, the cart control, or important mobile actions.

### Auto-dismiss (seconds)

Sets how long the message remains visible.

- If it is too short, visitors may not finish reading.
- If it is too long, the message may obstruct content.
- Allow more time for longer or more important messages.

## Preview

Use **Preview** to display different toast types in the theme editor. Choose **Off** to stop previewing.

Enable **Keep preview visible until dismissed** to keep the preview open while inspecting text, colors, position, and shadow.

::: info
Preview affects only the editor. Storefront duration is controlled by **Auto-dismiss**.
:::

## Shadow

| Setting | Purpose |
| --- | --- |
| **Toast shadow opacity** | Controls visibility; `0%` disables the shadow. |
| **Toast shadow horizontal offset** | Moves the shadow right with positive values and left with negative values. |
| **Toast shadow vertical offset** | Moves it down with positive values and up with negative values. |
| **Toast shadow blur** | Controls softness; higher values create a softer shadow. |

A low-opacity shadow with a slight downward offset and moderate blur usually provides clear separation without feeling heavy.

## Colors

Success, warning, error, and information colors come from **Theme settings > Colors > Feedback messages**. Edit the relevant color scheme to change toast backgrounds and foregrounds.

## Save and check

Before clicking **Save**, use Preview to confirm that the toast does not cover important content, text is readable, duration is sufficient, styling matches the theme, and the message fits on desktop and mobile.
