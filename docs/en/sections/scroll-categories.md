# Scrolling Categories

The **Scrolling Categories** section presents linked category or collection names in a continuously moving row. It provides a compact, dynamic way to direct customers to key storefront areas.

![Scrolling Categories settings and storefront preview](/images/en/scroll-categories.svg)

## Configure the section

Add one Item per category, select its destination, and arrange the order. Then configure:

- **Color Scheme**
- **Direction**
- **Speed**: Slow, Normal, or Fast
- **Horizontal spacing**
- **Height**
- **Content size**
- **Pause on hover**
- **Show divider**
- **Padding top / bottom**

Use short, consistent category names and avoid duplicate Items; the theme loops the existing sequence automatically. Each visible label must link to the matching published collection or page.

Start with Normal speed and enable hover pause. The movement must be slow enough to read on touch devices, which have no persistent hover state. Important conditions or legal information must also appear in a static area.

Height controls the moving band itself, while Padding controls separation from adjacent sections. Rebalance content size whenever height changes.

## Before publishing

Watch several complete cycles for gaps or jumps; test every speed and direction; verify all labels and links; check pause behavior, touch readability, mobile spacing, content size, height, divider contrast, keyboard focus, reduced-motion behavior, performance, and Custom CSS isolation. Save and test every category destination.
