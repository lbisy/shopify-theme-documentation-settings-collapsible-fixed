# Featured Products

The **Featured Products** section organizes multiple collections into tabs and displays their products as a grid or carousel. Use it for new arrivals, best sellers, or themed selections.

![Featured Products settings and storefront preview](/images/en/featured-products.svg)

## Add the section

Open a template, click **Add section > Featured Products**, then add one **Collection** block for each tab. Select the collections, drag blocks to reorder tabs, configure the section, and click **Save**.

## General and heading

**Color Scheme** controls the background, tabs, product information, navigation, and buttons. Check normal, selected, hover, and focus contrast.

**Heading size** sets tab-title size. Larger titles are prominent but can consume the available row when tabs are numerous or long. Use **Desktop tab header behavior** to control overflow; **Wrap to multiple lines** keeps all tab labels visible.

## Products Settings

- **Maximum items**: display limit for each selected collection.
- **Items per row on desktop**
- **Items per row on mobile**: **1** gives more room; **2** shows more products per screen.
- **Show Navigation**: controls browsing arrows or equivalent controls.
- **Show Pagination**: displays the current carousel position.
- **Show View All Button**: links the active tab to its collection page.

Products must be available and published to Online Store. If a collection contains fewer products than the maximum, only available products appear. Check long titles, prices, badges, and buttons whenever columns become narrow.

## Brand marquee

The decorative brand band repeats a name and moves it across the section.

- **Brand Name**: short rich text such as `Esen`.
- **Brand marquee direction**
- **Pause marquee on hover**
- **Brand marquee speed**: Slow, Normal, or Fast.
- **Brand marquee spacing** in `px`.
- **Brand marquee height** in `px`.

Start with Normal speed, keep text short, separate repetitions clearly, and enable hover pause. A large height strengthens the background effect but increases overall section height. Also review reduced-motion behavior.

## Padding

Set **Padding Top / Bottom** in `px`. Matching values are a good starting point, but reduce them when adjacent sections already provide substantial whitespace.

## Collection blocks

Each Collection block creates one tab. Add, replace, reorder, hide, duplicate, or remove blocks from the left panel. Put the most important collection first and avoid empty or substantially duplicated collections.

## Example

| Setting | Example |
| --- | --- |
| Heading size | `2XL` |
| Maximum items | `8` |
| Desktop / mobile columns | `4` / `1` |
| Navigation / Pagination / View All | Off / Off / Off |
| Tab behavior | `Wrap to multiple lines` |
| Brand Name | `Esen` |
| Direction / speed | Right to left / Normal |
| Pause on hover | On |
| Spacing / height | `64 px` / `380 px` |
| Padding | `32 px / 32 px` |

## Before publishing

Switch every tab and verify its name and products; test empty, short, and over-limit collections; verify product media, title, price, badges, and links; review every desktop and mobile column count; test tab wrapping, navigation, pagination, View All destinations, marquee direction, speed, spacing, pause, and responsive behavior; then use keyboard `Tab`, arrow keys, and `Enter` before saving.
