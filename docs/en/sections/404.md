# 404 Page

The 404 page appears when a visitor opens a link that does not exist or has been removed. In addition to the error message and return-to-home action, use the **Contact Us** settings to provide a way for visitors to ask for help.

## Open the 404 page settings

1. In Shopify admin, go to **Online Store > Themes**.
2. Find the current theme and click **Customize**.
3. Use the page selector at the top to open the **404 page**.
4. Select the corresponding 404 section in the template list.
5. Configure the section and click **Save**.

![404 page settings](/images/en/404.svg)

## Colors and form

### Color scheme

Select the global color scheme used by this area. To change the scheme itself, go to **Theme settings > Colors**.

### Form background color

Set the contact form background. Give the form enough visual separation from the surrounding page.

### Form text color

Set the color used for field text and supporting form messages. Ensure it has clear contrast with the form background.

## Image

### Image

Select an image for the contact area. Upload a new file or choose one from Shopify files or free images.

### Image alt text

Add a concise description for screen readers and for cases where the image cannot load. Leave it blank only when the image is purely decorative.

## Text content

### Heading

Set the contact-area heading, such as `Contact us` or `Need help?`. Basic rich-text formatting, including bold, italic, and links, is supported.

### Heading size

Choose the title size. Larger headings are more prominent, but check that they do not wrap excessively on mobile.

### Description

Add supporting text inviting visitors to describe a problem, ask about a product, or contact customer service.

### Button text

Set a short, clear submit label such as `Send` or `Submit message`.

### Success message

Set the confirmation shown after a successful submission, such as `Thank you for your message. We will reply as soon as possible.` Make it clear that the message was submitted.

### Watermark text

Set large decorative page text. Use a short brand name and ensure it does not reduce the readability of the heading, description, or form.

## Padding

- **Padding Top** controls space above this area.
- **Padding Bottom** controls space below it.

Higher values create a more open layout; lower values make it more compact. Review both desktop and mobile.

## Before saving

- Keep the 404 title and return-to-home action clearly visible.
- Ensure the image and watermark do not obscure primary content.
- Check form text and background contrast.
- Review heading and description wrapping on mobile.
- Use a clear submit-button label.
- Submit a test form and verify the Success message.
- Coordinate the vertical spacing with the Header and Footer.

::: tip Recommendation
The 404 page should help visitors quickly return to the store or contact support. Keep it focused on clear actions such as returning home, continuing to browse, or asking for help.
:::
