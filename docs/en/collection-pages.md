# Collection Pages

Collection pages group related products and help customers narrow choices with filtering and sorting. Create the collection and complete product data in Shopify admin before configuring headings, product count, filters, sorting, columns, and spacing.

![Collection page settings](/images/en/collection-page.svg)

## Create a collection

1. Go to **Shopify admin > Products > Collections**.
2. Create a collection and add its title and description.
3. Choose **Manual** membership or **Automated** conditions.
4. Add a collection image and default sort order as needed.
5. Complete search-engine information, publish to **Online Store**, and save.
6. Add the collection to a menu under **Content > Menus**.

Products must also be published and have accurate images, prices, inventory, product type, vendor, tags, variants, and metafields. These values affect cards and available filters.

## Open the collection template

Open **Online Store > Themes > Customize**, choose **Collections > Default collection**, then select the **Collection** section. Create and assign additional templates when collections need different layouts.

## Color and text

**Color Scheme** controls the section background, text, filter and sort controls, products, and decorative elements. Editing a global scheme affects every section using it.

- **Heading size**
- **Header**, such as `Products`
- **Description**
- **Subtitle size** for filters and sorting

Keep headings concise and preview subtitle controls on narrow screens.

## Rotating badge

**Rotate Badge Text** sets the outer circular copy; **Rotate Badge Percent** sets the center. Use short claims or percentages and ensure the badge does not overlap headings or controls.

## Products per page

Sets the maximum number before pagination. Larger values reduce page changes but increase loading and page length. Consider image size, total catalog size, and mobile performance.

## Filtering

- **Enable filtering**
- **Desktop filter layout**; **Drawer** opens filters from the side.
- **Visible filter values before show more**

Filters can include **Availability**, **Price**, and configured product attributes. Customers apply choices with **View Result**, whose number shows the matching count.

Configure available filters through the **Search & Discovery** app, not this section. Results depend on consistent product types, vendors, tags, variants, and metafields.

::: tip
Keep only filters that help customers decide. Use consistent names for common criteria such as price, availability, color, and size.
:::

## Sorting

**Enable sorting** displays **Sort by** options such as featured, best selling, title, price, or date. This switch controls customer access; set the collection's initial order in Shopify admin.

## Product-grid columns

- **Desktop product card columns**: **3 columns** or **4 columns**.
- **Mobile product card columns**: **1 column** or **2 columns**.

More columns make cards narrower. Check titles, pricing, badges, and buttons for crowding. Configure shared image ratio, border, radius, and shadow under **Theme settings > Product cards**.

## Padding

Use **Padding Top** and **Padding Bottom** in `px` to control separation from the header and following sections.

## Before publishing

- Verify collection and product publication, media, price, and inventory.
- Check heading, description, and badge on desktop and mobile.
- Test every filter, result count, clear action, drawer close, and **View Result**.
- Test all sort options.
- Test pagination beyond the page limit.
- Review desktop three/four-column and mobile one/two-column layouts.
- Confirm the menu opens the correct collection and save changes.
