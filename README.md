# Esen Theme Documentation

Bilingual online documentation project generated from the uploaded Shopify theme.

## Local preview

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repository into Vercel.
3. Vercel will automatically use `vercel.json`.
4. Build command: `npm run build`
5. Output directory: `docs/.vitepress/dist`

## Screenshots

Replace placeholder images in:

- `docs/public/images/en/`
- `docs/public/images/zh/`

Keep the existing filenames to avoid editing the Markdown files.
